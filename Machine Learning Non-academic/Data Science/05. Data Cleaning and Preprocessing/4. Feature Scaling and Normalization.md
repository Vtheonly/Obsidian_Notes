# 4. Feature Scaling and Normalization

Machine Learning algorithms that calculate **distance** (e.g., KNN, K-Means, SVM) perform poorly if features have different scales.
*   *Example:* Age (0-100) vs. Salary (20,000 - 100,000). Salary will dominate the distance calculation because the numbers are bigger.

## 4.1 Standardization (Z-Score Normalization)

Rescales data to have **Mean = 0** and **Standard Deviation = 1**.
*   **Formula:** $z = \frac{x - \mu}{\sigma}$
*   **Use Case:** When algorithms assume Gaussian distribution (Logistic Regression, SVM, Neural Networks). Preserves outliers.

```python
from sklearn.preprocessing import StandardScaler
scaler = StandardScaler()
df_scaled = scaler.fit_transform(df[['Age', 'Salary']])
```

## 4.2 Min-Max Normalization

Rescales data to a fixed range, usually **[0, 1]**.
*   **Formula:** $x_{norm} = \frac{x - x_{min}}{x_{max} - x_{min}}$
*   **Use Case:** Image processing (pixels 0-255), Neural Networks input layers. Sensitive to outliers.

```python
from sklearn.preprocessing import MinMaxScaler
scaler = MinMaxScaler()
df_norm = scaler.fit_transform(df[['Age', 'Salary']])
```

## 4.3 Robust Scaler

Scales data using **Median** and **IQR**.
*   **Formula:** $x_{robust} = \frac{x - Q2}{Q3 - Q1}$
*   **Use Case:** When the data contains significant outliers that you don't want to remove but also don't want influencing the scale.
