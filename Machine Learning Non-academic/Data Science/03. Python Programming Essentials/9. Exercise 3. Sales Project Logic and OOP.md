# 9. Exercise 3. Sales Project Logic and OOP

This exercise tests your ability to clean string data, use helper functions, and implement Object Oriented Programming to manage sales data.

## Scenario
You have a raw text file (`data.txt`) containing messy sales records.
Format: `Product, Price, Quantity`

**Task:**
1.  Read the file.
2.  Clean the lines.
3.  Parse them into objects.
4.  Calculate total revenue.

---

## Step 1: Helper Functions (`utils/helpers.py`)

We need robust functions to handle string manipulation.

```python
def clean_lines(lines):
    """
    Removes empty lines and strips whitespace.
    """
    cleaned = []
    for line in lines:
        if line.strip():  # If line is not empty
            cleaned.append(line.strip())
    return cleaned

def parse_line(line):
    """
    Converts string 'Laptop, 1000, 3' into tuple ('Laptop', 1000.0, 3)
    Handles errors gracefully.
    """
    try:
        parts = line.split(',')
        if len(parts) != 3:
            raise ValueError("Malformed line")
        
        name = parts[0].strip()
        price = float(parts[1].strip())
        qty = int(parts[2].strip())
        
        return name, price, qty
    except Exception as e:
        print(f"Skipping line '{line}': {e}")
        return None
```

## Step 2: The Product Class (`utils/product.py`)

```python
class Product:
    def __init__(self, name, price, quantity):
        self.name = name
        self.price = price
        self.quantity = quantity
    
    def get_total(self):
        return self.price * self.quantity
    
    def __str__(self):
        return f"{self.name}: ${self.get_total()}"
```

## Step 3: Main Execution (`scripts/main.py`)

```python
# Assume helper and product classes are imported
# RAW DATA SIMULATION
raw_data = [
    "Laptop, 1000, 3",
    "Mouse, 20, 50",
    "",  # Empty line
    "Monitor, 200, 5",
    "Corrupt Data Line"  # Error line
]

# 1. Clean
clean_data = clean_lines(raw_data)

# 2. Parse and Create Objects
products = []
for line in clean_data:
    parsed = parse_line(line)
    if parsed:
        name, price, qty = parsed
        products.append(Product(name, price, qty))

# 3. Calculate Revenue
total_revenue = sum([p.get_total() for p in products])

# Output
for p in products:
    print(p)
    
print(f"Total Revenue: ${total_revenue}")
```


**Key Takeaways:**
*   **Modularity:** Logic is separated into parsing vs. object modeling.
*   **Error Handling:** The program did not crash on the "Corrupt Data Line".
*   **OOP:** The `Product` object handles its own logic (`get_total`).
