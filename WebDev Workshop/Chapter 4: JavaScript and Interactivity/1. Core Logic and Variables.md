**Concept Overview:**
JavaScript is the **brain** of the web. While HTML defines the structure and CSS handles the appearance, JavaScript adds **logic and interactivity**. It can remember information, make decisions, react to user actions, and communicate with servers.

JavaScript is the only programming language that runs natively in every web browser. This makes it the undisputed language of the frontend.

---

## Variables — Storing Data

Variables are containers for storing data values. Think of them as labeled boxes where you put things.

**Modern JavaScript provides two keywords for declaring variables:**

```javascript
const siteName = "My Portfolio";    // Cannot be reassigned
let userScore = 0;                  // Can be reassigned later

userScore = 10;    // Valid — let allows reassignment
// siteName = "New Name";  // ERROR — const prevents reassignment
```

**The Golden Rule:** Always default to `const`. Only use `let` when you know the value **must change** later (like a score, a counter, or a toggle state).

**Avoid `var`:** The `var` keyword is the old way of declaring variables. It has confusing scoping behavior that leads to bugs. Modern JavaScript uses `const` and `let` exclusively.

## Data Types

JavaScript has several fundamental data types:

| Type | Example | Description |
|---|---|---|
| **String** | `"Hello"` or `'Hello'` | Text, wrapped in quotes |
| **Number** | `42`, `3.14` | Integers and decimals |
| **Boolean** | `true`, `false` | Logical yes/no values |
| **Array** | `["Apple", "Banana"]` | An ordered list of values |
| **Object** | `{ name: "Ali", age: 25 }` | A collection of key-value pairs |
| **Null** | `null` | Intentionally empty |
| **Undefined** | `undefined` | Declared but not yet assigned |

## Template Literals (Modern String Building)

Instead of clumsy string concatenation with `+`, use **template literals** (backtick strings):

```javascript
const userName = "Sara";
const greeting = `Welcome back, ${userName}! Your session started at ${new Date().toLocaleTimeString()}.`;
console.log(greeting);
// Output: "Welcome back, Sara! Your session started at 2:30:45 PM."
```

Template literals use backticks (`` ` ``) and `${}` to embed variables and expressions directly inside the string.

## Arrays — Working with Lists

```javascript
const fruits = ["Apple", "Banana", "Cherry"];

fruits.push("Date");          // Add to the end
fruits.pop();                 // Remove from the end
console.log(fruits.length);   // 3 — number of items
console.log(fruits[0]);       // "Apple" — arrays are 0-indexed

// Loop through an array
fruits.forEach(function(fruit) {
    console.log(fruit);
});
```
