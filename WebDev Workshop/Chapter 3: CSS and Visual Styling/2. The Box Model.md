**Crucial Background Knowledge:**
In CSS, **everything is a box**. Every single element вҖ” every image, every paragraph, every button, every heading вҖ” is wrapped in an invisible rectangular box. Understanding the Box Model is the single most important concept in CSS layout.

**The Four Layers of the Box Model (from inside out):**

1.  **Content:** The actual text, image, or element itself. Its size is controlled by `width` and `height`.
2.  **Padding:** Transparent space **inside** the box, between the content and the border. It pushes the content inward.
3.  **Border:** The visible line wrapping around the padding and content. You can control its thickness, style, and color.
4.  **Margin:** Transparent space **outside** the box. It pushes the element away from other elements around it.

**CSS Code Example:**
```css
.card {
    width: 300px;
    padding: 20px;
    border: 2px solid #333;
    margin: 30px;
}
```

**The `box-sizing` Problem and Fix:**
By default, CSS uses `content-box` sizing. This means if you set `width: 300px` and add `padding: 20px`, the **total visible width** becomes 300 + 20 + 20 = **340px**. This is extremely confusing and leads to layout bugs.

**The Fix:** Always add this rule at the very top of your CSS file:
```css
*, *::before, *::after {
    box-sizing: border-box;
}
```
With `border-box`, the padding and border are included **inside** the specified width. So `width: 300px` means the element will be exactly 300px wide, no matter how much padding or border you add. The content area shrinks to make room.

**Shorthand Properties:**
```css
/* Individual sides: top, right, bottom, left (clockwise) */
margin: 10px 20px 10px 20px;

/* Two values: vertical, horizontal */
padding: 10px 20px;

/* One value: all sides */
margin: 15px;
```

**Margin Collapse:**
An important quirk: when two vertical margins touch (e.g., the bottom margin of one element meets the top margin of the next), they **collapse** into a single margin equal to the larger value. This only happens vertically, never horizontally.
