**Concept Overview:**
Before modern CSS layout tools, placing items side-by-side was a nightmare involving `float` hacks and complicated clearfixes. Now, we have two powerful, purpose-built systems: **Flexbox** and **Grid**.

---

## Flexbox (1-Dimensional Layout)

Flexbox is perfect for arranging items in a **single direction** — either a row or a column.

**How to Use It:**
Apply `display: flex;` to a **parent container**, and all its direct children become flex items.

```css
.container {
    display: flex;
    justify-content: center;   /* Horizontal alignment */
    align-items: center;       /* Vertical alignment */
    gap: 20px;                 /* Space between items */
}
```

**Key Flexbox Properties (on the parent):**

| Property | Values | Purpose |
|---|---|---|
| `flex-direction` | `row`, `column` | Sets the main axis direction |
| `justify-content` | `flex-start`, `center`, `flex-end`, `space-between`, `space-around`, `space-evenly` | Aligns items along the main axis |
| `align-items` | `flex-start`, `center`, `flex-end`, `stretch` | Aligns items along the cross axis |
| `flex-wrap` | `nowrap`, `wrap` | Allows items to wrap to next line |
| `gap` | Any length (`10px`, `1rem`) | Adds space between items |

**Key Flexbox Properties (on the children):**

| Property | Purpose |
|---|---|
| `flex: 1;` | Makes the item grow to fill available space equally |
| `flex-grow` | How much the item should grow relative to siblings |
| `flex-shrink` | How much the item should shrink when space is tight |
| `align-self` | Overrides the parent's `align-items` for this specific item |

## CSS Grid (2-Dimensional Layout)

Grid is perfect for complex layouts with **both rows AND columns** at the same time — like a photo gallery, a dashboard, or a full page layout.

```css
.grid-container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;   /* 3 equal columns */
    grid-template-rows: auto auto;         /* 2 rows that auto-size */
    gap: 20px;
}
```

**Key Grid Terminology:**
*   `fr` (fraction unit): Divides available space proportionally. `1fr 2fr` means the second column is twice as wide as the first.
*   `grid-template-columns`: Defines the number and width of columns.
*   `grid-template-rows`: Defines the number and height of rows.
*   `grid-column: 1 / 3;`: Makes an item span from column line 1 to column line 3 (i.e., spanning 2 columns).

---

## When to Use Which?

| Scenario | Use |
|---|---|
| Navigation bar (items in a row) | **Flexbox** |
| Centering a single element | **Flexbox** |
| Photo gallery (rows + columns) | **Grid** |
| Full page layout (header, sidebar, main, footer) | **Grid** |
| A simple list of cards | **Flexbox** |
| A complex dashboard | **Grid** |

**Pro Tip:** You can nest them. Use Grid for the overall page layout and Flexbox inside individual components (like a navigation bar within a Grid header).
