**Concept Overview:**
HTML (HyperText Markup Language) is not a programming language; it is a **markup language**. This is an important distinction. A programming language (like JavaScript or Python) can make decisions and perform calculations. A markup language simply tells the browser **what kind of content** is on the page.

Think of it this way: HTML is like an architect's blueprint. It says "a wall goes here, a window goes there, and a door is in between." It defines the **structure** but says nothing about the color of the walls or what happens when you open the door.

**The Anatomy of an HTML Element:**
Most elements require an opening tag, the content, and a closing tag.

```html
<a href="https://example.com">Click Here</a>
```

*   `<a>`: The **opening tag**. The `a` stands for "anchor" (a link).
*   `href="https://example.com"`: An **attribute** providing extra information (the destination URL).
*   `Click Here`: The **content** the user sees on the page.
*   `</a>`: The **closing tag** (indicated by the forward slash `/`).

**Self-Closing Tags:**
Some elements do not have content between them, so they close themselves:

```html
<img src="cat.jpg" alt="A fluffy orange cat">
<br>
<hr>
<input type="text">
```

*   `<img>`: Displays an image. Requires `src` (source path) and `alt` (alternative text).
*   `<br>`: A line break (moves text to the next line).
*   `<hr>`: A horizontal rule (draws a line across the page).
*   `<input>`: A form input field.

**Important Tip:** Always include the `alt` (alternative text) attribute for images. It serves two critical purposes:
1.  **Accessibility:** Visually impaired users who use screen readers will hear the `alt` text read aloud.
2.  **SEO:** Search engines like Google cannot "see" images. They rely on the `alt` text to understand what the image contains, which helps your page rank higher in search results.

**Common HTML Tags You Must Know:**

| Tag | Purpose | Example |
|---|---|---|
| `<h1>` to `<h6>` | Headings (h1 is largest) | `<h1>Main Title</h1>` |
| `<p>` | Paragraph of text | `<p>Hello world</p>` |
| `<a>` | Hyperlink | `<a href="url">Link</a>` |
| `<img>` | Image | `<img src="photo.jpg" alt="desc">` |
| `<ul>` / `<ol>` | Unordered / Ordered list | `<ul><li>Item</li></ul>` |
| `<div>` | Generic container (block) | `<div>Content</div>` |
| `<span>` | Generic container (inline) | `<span>Text</span>` |
| `<strong>` | Bold text (semantic) | `<strong>Important</strong>` |
| `<em>` | Italic text (semantic) | `<em>Emphasis</em>` |
