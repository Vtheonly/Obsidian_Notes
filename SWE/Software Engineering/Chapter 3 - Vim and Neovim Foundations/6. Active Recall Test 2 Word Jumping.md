### A1 — Three Primary Word Motions

- `w` — move forward to the start of the next word. Mnemonic: **w**ord.
- `e` — move forward to the end of the current word (or next word if already at end). Mnemonic: **e**nd.
- `b` — move backward to the start of the current word (or previous if already at start). Mnemonic: **b**ack.

### A2 — Word vs WORD

A **word** is a sequence of letters/digits/underscores, OR a sequence of other non-blank characters. Punctuation breaks words.

A **WORD** is a sequence of any non-blank characters. Only whitespace breaks WORDs.

Example: in `foo.bar`, there are 3 words (`foo`, `.`, `bar`) but 1 WORD (`foo.bar`).

### A3 — Uppercase Variants

`W`, `E`, `B` behave like `w`, `e`, `b` but operate on WORDs instead of words. They skip over punctuation.

### A4 — Lands On

- `w` lands on the **first character** of the next word.
- `e` lands on the **last character** of the current word (or the next word if already at the last character).

### A5 — Operators on Words

- `dw` — delete from cursor up to (but not including) the start of the next word. Includes trailing whitespace.
- `de` — delete from cursor up to and including the end of the current word. Does NOT include trailing whitespace.

### A6 — Text Objects

- `iw` — inner word. Selects the word under the cursor without surrounding whitespace.
- `aw` — a word. Selects the word plus one surrounding whitespace (usually a trailing space).

`daw` deletes the whole word plus trailing whitespace, regardless of where the cursor is in the word. `dw` only deletes from the cursor forward, so if the cursor is in the middle of the word, `dw` does not delete the whole word.

### A7 — Change Word

Two ways:

1. `ciw` — change inner word. Deletes the word and enters Insert mode.
2. `caw` — change a word. Deletes the word and trailing whitespace, enters Insert mode.

(`cw` is also common but behaves like `ce` — it changes from the cursor to the end of the current word, which is not the whole word if the cursor is mid-word.)

### A8 — End of Word Append

Press `ea`. `e` moves to the end of the current word; `a` enters Insert mode one character to the right (append).

(Or `ei` if you want to be on the last character in Insert mode rather than after it, but `ea` is the more common intent.)

Actually, careful: `e` lands ON the last character. `a` after `e` puts you in Insert mode AFTER that last character. So `ea` lets you type after the last character of the word. If you want to type before the last character, use `ei` (or just `e` then `i`).

### A9 — Code With Dots

In `obj.method.property`:

- Words: `obj`, `.`, `method`, `.`, `property` — **5 words**.
- WORDs: `obj.method.property` — **1 WORD**.

`w` stops at each dot; `W` skips the entire expression.

### A10 — Renaming a Variable

1. Place the cursor on the variable.
2. Press `*` to search forward for the next occurrence.
3. Press `ciw` to change the word — type the new name.
4. Press `Esc` to return to Normal mode.
5. Press `n` to go to the next occurrence.
6. Press `.` to repeat the change.
7. Repeat `n` and `.` until done.

The `.` command repeats the last change, which in this case is `ciw` followed by the typed text.
