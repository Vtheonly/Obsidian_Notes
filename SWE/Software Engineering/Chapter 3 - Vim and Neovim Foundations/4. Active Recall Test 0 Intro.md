### A1 — Modes

1. **Normal** — navigate and manipulate text. Default at startup.
2. **Insert** — type text into the buffer.
3. **Visual** — select a region of text.
4. **Command-line** — type Ex commands like `:w`, `:q`.

### A2 — Default Mode

Normal mode.

### A3 — Entering Insert Mode

| Key | Cursor lands |
| --- | --- |
| `i` | Insert at current cursor |
| `a` | Append — one character right of cursor |
| `I` | Beginning of line (first non-blank) |
| `A` | End of line |
| `o` | Open new line below |
| `O` | Open new line above |

### A4 — Returning to Normal Mode

`Esc`. Synonyms: `Ctrl-[`, `Ctrl-c` (Neovim); some users remap `jk` or `kj` to `Esc`.

### A5 — Saving and Quitting

| Command | Action |
| --- | --- |
| `:w` | Write (save) the buffer to disk. |
| `:q` | Quit. Fails if unsaved changes. |
| `:wq` | Write and quit. |
| `:x` | Write (only if changed) and quit. |
| `:q!` | Quit without saving (force). |

Use `:q!` when you have made changes you do not want to keep. Use `:x` when you want to save only if something changed (slightly faster than `:wq`).

### A6 — Two-Letter Shortcuts

- `ZZ` — save and quit (same as `:x`).
- `ZQ` — quit without saving (same as `:q!`).

They are faster because they skip the command-line mode (`:`) — two keystrokes versus four.

### A7 — Stuck in Vim

1. Press `Esc`.
2. Type `:q!`.
3. Press `Enter`.

### A8 — Opening a File

```bash
vim notes.txt
```

If the file does not exist, Vim creates an empty buffer for it. The file is actually created on disk only when you save.

### A9 — Configuration

- Vim: `~/.vimrc` (Vimscript).
- Neovim: `~/.config/nvim/init.vim` (Vimscript) or `~/.config/nvim/init.lua` (Lua).

### A10 — Vim vs Neovim

Any three of:

- Default config file location.
- Neovim supports Lua for configuration; Vim uses Vimscript only.
- Neovim has a first-class built-in terminal; Vim's is limited.
- Neovim has built-in LSP support; Vim requires plugins.
- Neovim has a refactored, tested C codebase; Vim's is older.
- Neovim has first-class async jobs; Vim's are limited (Vim 8+).
