## Questions

### Q1 — The Four Direction Keys

Which four keys move the cursor one character at a time, and which direction does each move?

### Q2 — Arrow Keys

Why should you avoid using the arrow keys in Vim, even though they work?

### Q3 — Counts

How do you move 7 characters to the right in a single command? How do you move down 4 lines?

### Q4 — Line Extremes

Which key moves to the first character of the line? Which moves to the first non-blank character? Which moves to the last character?

### Q5 — File Extremes

How do you jump to the first line of the file? How do you jump to the last line? How do you jump to line 42?

### Q6 — Viewport Motion

What do `H`, `M`, and `L` do? What do `Ctrl-d` and `Ctrl-u` do? What does `zz` do?

### Q7 — Find on Line

### Q8 — Search

How do you search forward for "function"? How do you repeat the search in the same direction? In the opposite direction? How do you search for the word under the cursor without typing it?

### Q9 — Marks

How do you set a mark named `a`? How do you jump back to it? How do you jump to the position before the last jump?

### Q10 — Operator Plus Motion

What does `d$` do? What does `d0` do? What does `dt)` do? What does `df)` do?

### A1 — The Four Direction Keys

- `h` — left
- `j` — down
- `k` — up
- `l` — right

### A2 — Arrow Keys

Arrow keys force you to leave the home row, which slows you down. They also prevent you from developing muscle memory for `hjkl`. Many users disable them in their config to enforce the habit.

### A3 — Counts

- 7 characters right: `7l`
- 4 lines down: `4j`

Counts work with nearly every Vim motion and command.

### A4 — Line Extremes

- `0` — first character of the line.
- `^` — first non-blank character of the line.
- `$` — last character of the line.

(`g_` is the last non-blank character.)

### A5 — File Extremes

- First line: `gg`
- Last line: `G`
- Line 42: `:42<CR>` or `42G`

### A6 — Viewport Motion

- `H` — go to the **H**ighest (top) visible line.
- `M` — go to the **M**iddle visible line.
- `L` — go to the **L**owest (bottom) visible line.
- `Ctrl-d` — scroll down half a page.
- `Ctrl-u` — scroll up half a page.
- `zz` — center the current line on screen.

### A7 — Find on Line

- Next comma: `f,`
- Repeat the find: `;`
- Previous comma: `F,` (capital F to search backward)

(`t,` and `T,` move until just before the comma, rather than onto it.)

### A8 — Search

- Forward search: `/function<CR>`
- Repeat same direction: `n`
- Repeat opposite direction: `N`
- Word under cursor: `*` (forward) or `#` (backward)

### A9 — Marks

- Set mark `a`: `ma`
- Jump to mark `a`: `` `a `` (backtick a) — line and column. Or `'a` (single quote a) — first non-blank of the line.
- Jump back before the last jump: `` `` `` (two backticks)

### A10 — Operator Plus Motion

- `d$` — delete from cursor to end of line.
- `d0` — delete from cursor to beginning of line.
- `dt)` — delete up to (but not including) the next `)`.
- `df)` — delete up to and including the next `)`.
