# 2. IntelliJ and JetBrains IDEs

> **Tags:** #intellij #jetbrains #ide #editors #java #kotlin

JetBrains IDEs (IntelliJ IDEA, WebStorm, PyCharm, Rider, CLion, GoLand, RubyMine, PhpStorm) are the most powerful IDEs for statically-typed languages. They share a common platform — the IntelliJ Platform — so most features work identically across all of them.

---

## 2.1 Why JetBrains IDEs Are Different

JetBrains IDEs perform **deep semantic analysis** of your code. They build a complete model of your project's types, dependencies, and call graph. This enables features that text-editor-based IDEs cannot match:

- **Refactoring** that actually works (rename a method and it updates every caller, including in other files and languages).
- **Smart completion** that filters by expected type, not just by prefix.
- **Find usages** that understands inheritance, generics, and dynamic dispatch.
- **Inspections** that detect not just syntax errors but semantic issues (e.g., "this code is unreachable").

## 2.2 Search and Navigation

| Shortcut | Action |
| --- | --- |
| `Double Shift` | Search Everywhere (files, classes, symbols, actions, settings). |
| `Ctrl+N` | Find class by name. |
| `Ctrl+Shift+N` | Find file by name. |
| `Ctrl+Shift+A` | Find Action (like Command Palette). |
| `Ctrl+B` / `Ctrl+Click` | Go to declaration. |
| `Ctrl+Alt+B` | Go to implementation (for interfaces). |
| `Ctrl+Shift+B` | Go to type declaration. |
| `Alt+F7` | Find usages. |
| `Ctrl+Alt+F7` | Show usages (popup). |
| `Ctrl+E` | Recent files. |
| `Ctrl+Shift+E` | Recent locations. |
| `Ctrl+Alt+Left` / `Ctrl+Alt+Right` | Navigate back / forward. |
| `Ctrl+F12` | File structure (symbols in current file). |
| `Ctrl+H` | Type hierarchy. |
| `Ctrl+Shift+H` | Method hierarchy. |

The **Search Everywhere** (double Shift) is the JetBrains equivalent of VS Code's Command Palette. It searches files, classes, symbols, actions, and settings simultaneously.

## 2.3 Refactoring

JetBrains has the best refactoring tooling of any IDE. Every refactor is safe and updates all references.

| Shortcut | Action |
| --- | --- |
| `Shift+F6` | Rename (class, method, variable, file). |
| `Ctrl+Alt+M` | Extract method. |
| `Ctrl+Alt+V` | Extract variable. |
| `Ctrl+Alt+F` | Extract field. |
| `Ctrl+Alt+C` | Extract constant. |
| `Ctrl+Alt+P` | Extract parameter. |
| `F6` | Move (class, method). |
| `Ctrl+Alt+N` | Inline (method, variable). |
| `Ctrl+F6` | Change signature. |
| `Ctrl+Alt+Shift+T` | Refactor this (menu of all refactors). |

## 2.4 Code Completion

JetBrains has three levels of completion:

1. **Basic completion** (`Ctrl+Space`): suggests names of classes, methods, fields, and keywords in the current context.
2. **Smart completion** (`Ctrl+Shift+Space`): filters suggestions by expected type. If you are assigning to a `String` variable, only `String`-returning expressions are suggested.
3. **Statement completion** (`Ctrl+Shift+Enter`): completes the current statement — adds semicolons, closing braces, parentheses.

---

## 2.5 Inspections and Quick Fixes

JetBrains runs hundreds of **inspections** — static analysis checks that highlight potential issues in real time. Press `Alt+Enter` on a highlighted line to see **quick fixes**:

- Convert verbose code to a lambda.
- Add missing annotations.
- Simplify boolean expressions.
- Extract a magic number to a constant.
- Add a missing import.

`Alt+Enter` is the most-used shortcut in JetBrains IDEs. When you see a yellow warning underline, press `Alt+Enter` and let the IDE fix it.

## 2.6 Live Templates

Live templates are snippets on steroids. Type an abbreviation and press Tab to expand:

| Abbreviation | Expands to |
| --- | --- |
| `sout` | `System.out.println();` (Java) |
| `psvm` | `public static void main(String[] args)` |
| `fori` | `for (int i = 0; i < ; i++) {}` |
| `iter` | Iterate over a collection. |
| `ifn` | `if (var == null) {}` |

Define custom templates in Settings → Editor → Live Templates.

## 2.7 Version Control Integration

JetBrains has first-class Git integration:

- **Commit tool window** (`Alt+0`): stage and commit changes with a diff view.
- **Log tab**: visual history with branch graph.
- **Conflict resolution**: three-way merge tool built in.
- **Annotate** (blame): right-click a file → Git → Annotate.
- **Stash and shelves**: shelves are JetBrains-specific local changes storage.

---

## 2.8 Debugger

JetBrains debuggers support:

- Breakpoints (line, method, exception).
- Conditional breakpoints.
- Watch expressions.
- Evaluate expression (`Alt+F8`).
- Drop frame (go back to the start of the current method).
- Remote debugging (attach to a JVM, Python process, etc.).

---

## 2.9 Plugins

JetBrains has a large plugin ecosystem. Notable plugins:

- **IdeaVim** — Vim emulation for JetBrains.
- **.ignore** — manage `.gitignore` files.
- **Key Promoter X** — shows shortcuts for actions you perform with the mouse.
- **Rainbow Brackets** — color-matched brackets.
- **GitToolBox** — extra Git features (inline blame, status).

---

## 2.10 Key Takeaways

- JetBrains IDEs perform deep semantic analysis, enabling safe refactoring and smart completion.
- Double Shift = Search Everywhere; `Ctrl+Shift+A` = Find Action.
- Refactoring shortcuts: `Shift+F6` (rename), `Ctrl+Alt+M` (extract method), `F6` (move).
- `Alt+Enter` is the magic key for quick fixes.
- Three completion levels: basic (`Ctrl+Space`), smart (`Ctrl+Shift+Space`), statement (`Ctrl+Shift+Enter`).
- Live templates (snippets) save typing for common patterns.
