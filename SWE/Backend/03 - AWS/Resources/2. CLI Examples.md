## 1. Identity and Configuration

```bash
# Who am I?
aws sts get-caller-identity

# List profiles
aws configure list-profiles

# Show current configuration
aws configure list

# Get the account ID
aws sts get-caller-identity --query 'Account' --output text

# Get the region
aws configure get region
```

## 2. S3 Patterns

### 2.1 Upload a Directory Recursively

```bash
aws s3 sync ./local-dir s3://my-bucket/prefix/ \
  --exclude "*.tmp" \
  --exclude ".git/*" \
  --delete
```

### 2.2 Generate a Presigned URL

```bash
aws s3 presign s3://my-bucket/file.pdf --expires-in 3600
```

### 2.3 Get the Size of a Bucket

```bash
aws s3 ls s3://my-bucket --recursive --summarize | grep "Total Size"
```

### 2.4 Copy Between Buckets (Different Accounts)

```bash
aws s3 cp s3://source-bucket/file.txt s3://dest-bucket/file.txt \
  --source-region us-east-1 --region us-west-2
```

### 2.5 List Buckets by Region

```bash
for region in us-east-1 us-west-2 eu-west-1; do
  echo "Region: $region"
  aws s3api list-buckets --region $region --query 'Buckets[*].Name' --output text
done
```

## 3. EC2 Patterns

### 3.1 Find Running Instances

```bash
aws ec2 describe-instances \
  --filters "Name=instance-state-name,Values=running" \
  --query 'Reservations[*].Instances[*].[InstanceId,Tags[?Key==`Name`].Value | [0],State.Name]' \
  --output table
```

### 3.2 Start, Stop, Terminate

```bash
aws ec2 start-instances --instance-ids i-12345 i-67890
aws ec2 stop-instances --instance-ids i-12345
aws ec2 terminate-instances --instance-ids i-12345
```

### 3.3 Get the Latest Amazon Linux AMI

```bash
aws ssm get-parameters \
  --names /aws/service/ami-amazon-linux-latest/al2023-ami-kernel-6.1-x86_64 \
  --query 'Parameters[0].Value' --output text
```

### 3.4 List Security Groups

```bash
aws ec2 describe-security-groups \
  --query 'SecurityGroups[*].[GroupId,GroupName,Description]' \
  --output table
```

## 4. IAM Patterns

### 4.1 List Users with Their Groups

```bash
for user in $(aws iam list-users --query 'Users[*].UserName' --output text); do
  groups=$(aws iam list-groups-for-user --user-name $user --query 'Groups[*].GroupName' --output text | tr '\t' ',')
  echo "$user: $groups"
done
```

### 4.2 Find All Access Keys Older Than 90 Days

```bash
for user in $(aws iam list-users --query 'Users[*].UserName' --output text); do
  aws iam list-access-keys --user-name $user --query 'AccessKeyMetadata[*].[AccessKeyId,CreateDate]' --output text
done
```

### 4.3 Assume a Role

```bash
aws sts assume-role --role-arn arn:aws:iam::123456789012:role/MyRole --role-session-name MySession
```

## 5. DynamoDB Patterns

### 5.1 Query with Pagination

```bash
LAST_KEY=""
while true; do
  if [ -z "$LAST_KEY" ] || [ "$LAST_KEY" = "null" ]; then
    response=$(aws dynamodb scan --table-name my-table --max-items 100)
  else
    response=$(aws dynamodb scan --table-name my-table --max-items 100 --exclusive-start-key "$LAST_KEY")
  fi
  
  echo "$response" | jq '.Items'
  LAST_KEY=$(echo "$response" | jq -c '.LastEvaluatedKey')
  
  if [ "$LAST_KEY" = "null" ] || [ -z "$LAST_KEY" ]; then break; fi
done
```

### 5.2 Backup a Table to S3

```bash
aws dynamodb export-table-to-point-in-time \
  --table-arn arn:aws:dynamodb:us-east-1:123456789012:table/my-table \
  --s3-bucket my-backup-bucket \
  --s3-prefix dynamodb-backups/my-table/ \
  --export-format DYNAMODB_JSON
```

## 6. Lambda Patterns

### 6.1 Invoke Synchronously

```bash
aws lambda invoke --function-name my-func \
  --payload '{"key": "value"}' \
  --cli-binary-format raw-in-base64-out \
  output.json
```

### 6.2 Invoke Asynchronously

```bash
aws lambda invoke --function-name my-func \
  --invocation-type Event \
  --payload '{"key": "value"}' \
  --cli-binary-format raw-in-base64-out \
  output.json
```

### 6.3 Get Function Configuration

```bash
aws lambda get-function-configuration --function-name my-func
```

### 6.4 Update Environment Variables

```bash
aws lambda update-function-configuration --function-name my-func \
  --environment "Variables={LOG_LEVEL=debug,TABLE_NAME=my-table}"
```

## 7. CloudWatch Logs Patterns

### 7.1 Tail Logs Live

```bash
aws logs tail /aws/lambda/my-func --follow
```

### 7.2 Filter Logs by Pattern

```bash
aws logs filter-log-events \
  --log-group-name /aws/lambda/my-func \
  --filter-pattern "ERROR" \
  --start-time $(date -d '1 hour ago' +%s)000
```

### 7.3 Run an Insights Query

```bash
QUERY_ID=$(aws logs start-query \
  --log-group-name /aws/lambda/my-func \
  --start-time $(date -d '1 hour ago' +%s) \
  --end-time $(date +%s) \
  --query-string 'fields @timestamp, @message | filter @message like /ERROR/ | limit 20' \
  --query 'queryId' --output text)

sleep 5
aws logs get-query-results --query-id $QUERY_ID
```

## 8. Cost Patterns

### 8.1 Get Current Month's Cost by Service

```bash
aws ce get-cost-and-usage \
  --time-period Start=$(date -d "$(date +%Y-%m-01)" +%Y-%m-%d),End=$(date -d "tomorrow" +%Y-%m-%d) \
  --granularity MONTHLY \
  --metrics BlendedCost \
  --group-by Type=DIMENSION,Key=SERVICE
```

### 8.2 Get Cost by Tag

```bash
aws ce get-cost-and-usage \
  --time-period Start=2024-01-01,End=2024-02-01 \
  --granularity MONTHLY \
  --metrics BlendedCost \
  --group-by Type=DIMENSION,Key=SERVICE Type=TAG,Key=Team
```

## 9. Common Pipeline Patterns

### 9.1 Get All Instance IDs and Stop Them

```bash
aws ec2 describe-instances \
  --filters "Name=instance-state-name,Values=running" \
  --query 'Reservations[*].Instances[*].InstanceId' \
  --output text | tr '\t' '\n' | xargs -n 1 aws ec2 stop-instances --instance-ids
```

### 9.2 List All S3 Buckets and Their Sizes

```bash
for bucket in $(aws s3api list-buckets --query 'Buckets[*].Name' --output text); do
  size=$(aws s3 ls s3://$bucket --recursive --summarize 2>/dev/null | grep "Total Size" | awk '{print $3}')
  echo "$bucket: $size"
done
```

### 9.3 Find All Public S3 Buckets

```bash
for bucket in $(aws s3api list-buckets --query 'Buckets[*].Name' --output text); do
  policy=$(aws s3api get-bucket-policy-status --bucket $bucket 2>/dev/null)
  if echo "$policy" | jq -e '.PolicyStatus.IsPublic' > /dev/null 2>&1; then
    echo "PUBLIC: $bucket"
  fi
done
```

## 10. Useful `--query` Patterns

### 10.1 Get the First Item

```bash
--query 'Buckets[0].Name'
```

### 10.2 Get a Specific Field from All Items

```bash
--query 'Buckets[*].Name'
```

### 10.3 Filter

```bash
--query 'Users[?UserName==`alice`]'
```

### 10.4 Multi-Field

```bash
--query 'Reservations[*].Instances[*].[InstanceId,State.Name]'
```

### 10.5 Sort

```bash
--query 'sort_by(Users, &CreateTime)'
```
