## 2. Common CLI Commands Cheat Sheet

### S3

```bash
aws s3 ls                                     # list buckets
aws s3 ls s3://my-bucket/                     # list objects
aws s3 mb s3://my-bucket                      # create bucket
aws s3 cp file.txt s3://my-bucket/            # upload
aws s3 cp s3://my-bucket/file.txt .           # download
aws s3 sync ./dir s3://my-bucket/dir/         # sync directories
aws s3 presign s3://my-bucket/file.txt        # presigned URL
aws s3 rm s3://my-bucket/file.txt             # delete
aws s3 rb s3://my-bucket                      # delete bucket
```

### EC2

```bash
aws ec2 describe-instances                   # list instances
aws ec2 run-instances --image-id ami-... --instance-type t3.micro
aws ec2 start-instances --instance-ids i-...
aws ec2 stop-instances --instance-ids i-...
aws ec2 terminate-instances --instance-ids i-...
```

### IAM

```bash
aws iam list-users
aws iam create-user --user-name alice
aws iam create-access-key --user-name alice
aws iam attach-user-policy --user-name alice --policy-arn arn:aws:iam::aws:policy/AmazonS3ReadOnlyAccess
aws sts get-caller-identity                  # who am I?
```

### DynamoDB

```bash
aws dynamodb list-tables
aws dynamodb create-table --table-name Users \
  --attribute-definitions AttributeName=id,AttributeType=N \
  --key-schema AttributeName=id,KeyType=HASH \
  --billing-mode PAY_PER_REQUEST
aws dynamodb put-item --table-name Users --item '{"id":{"N":"1"},"name":{"S":"Alice"}}'
aws dynamodb get-item --table-name Users --key '{"id":{"N":"1"}}'
aws dynamodb scan --table-name Users
aws dynamodb query --table-name Users --key-condition-expression "id = :id" --expression-attribute-values '{":id":{"N":"1"}}'
```

### Lambda

```bash
aws lambda list-functions
aws lambda create-function --function-name my-func --runtime python3.11 --handler index.handler --role arn:... --zip-file fileb://function.zip
aws lambda invoke --function-name my-func output.json
aws lambda update-function-code --function-name my-func --zip-file fileb://function.zip
```

### SQS

```bash
aws sqs create-queue --queue-name my-queue
aws sqs send-message --queue-url https://sqs.us-east-1.amazonaws.com/.../my-queue --message-body "Hello"
aws sqs receive-message --queue-url ... --wait-time-seconds 20
aws sqs delete-message --queue-url ... --receipt-handle ...
```

### Logs

```bash
aws logs describe-log-groups
aws logs tail /aws/lambda/my-func --follow
aws logs filter-log-events --log-group-name /aws/lambda/my-func --filter-pattern "ERROR"
```
