### 4.3 Diagnostics, Verification, & File Management

Run these commands from the **Privileged EXEC Mode** (`Router#`) to verify configurations and troubleshoot network connectivity:

* **Show Running Configuration:**
  ```ios
  R1# show running-config
  ```
  Displays the current configuration active in RAM.

* **Show Device Hardware & IOS Version:**
  ```ios
  R1# show version
  ```
  Displays uptime, hardware interfaces, and IOS image details.

* **Show IP Interfaces:**
  ```ios
  R1# show ip interface brief
  ```
  Displays interface status, IP assignments, and protocol states in a table.

* **Show Routing Table:**
  ```ios
  R1# show ip route
  ```
  Displays active routes, administrative distances, metrics, and exit interfaces.

* **Show MAC Address Table (Switches):**
  ```ios
  Switch# show mac-address-table
  ```
  Displayslearned MAC addresses, their associated ports, and access VLANs.

* **Save Active Configuration:**
  ```ios
  R1# copy running-config startup-config
  -- Alternative short command:
  R1# write
  ```
  Saves the active configuration from volatile RAM to non-volatile NVRAM so configurations persist across reboots.

* **Revert to Saved Configuration:**
  ```ios
  R1# copy startup-config running-config
  ```
  Restores configuration parameters from NVRAM to RAM, overwriting any unsaved changes.
