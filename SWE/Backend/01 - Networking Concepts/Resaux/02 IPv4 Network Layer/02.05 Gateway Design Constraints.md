### 2.5 Gateway Design Constraints
* **Subnet Dependency:** A host's default gateway **must** reside within the same local subnet as the host itself.
* **IP Allocation Best Practices:** The default gateway is typically assigned either the **first usable IP** (e.g., `.1`) or the **last usable IP** (e.g., `.254`) in the subnet.
* *Exam Application:* If a host with IP `143.27.172.101/18` (usable range: `143.27.128.1`–`143.27.191.254`) is configured with a default gateway of `143.27.105.1`, explain why this configuration is invalid. The gateway address `143.27.105.1` lies in subnet `143.27.64.0/18` (range: `143.27.64.1`–`143.27.127.254`), making it unreachable without an intermediate router.
