### 2.2 Fixed-Length Subnet Masking (FLSM)

FLSM divides a network block into equal-sized subnets by borrowing a fixed number of bits from the host portion.

#### Mathematical Formulation

* **Number of borrowed subnet bits ($n$):**
  $$2^n \ge \text{Number of required subnets}$$

* **Number of remaining host bits ($h$):**
  $$h = 32 - (\text{Default Prefix Length} + n)$$

* **Total usable hosts per subnet ($N_{\text{hosts}}$):**
  $$N_{\text{hosts}} = 2^h - 2$$

#### Comprehensive Calculation Walkthrough
*Given:* Network block `184.65.94.20` with custom subnet mask `255.255.240.0`.

1. **Class and Base Network Identification:**
   * First octet = `184` (Class B, default prefix `/16`).
   * Default classful network: `184.65.0.0/16`.

2. **Parsing the Custom Subnet Mask:**
   * Custom Mask: `255.255.240.0`
   * Third Octet in Binary: `240` = `11110000`
   * Total network + subnet prefix length = $8 + 8 + 4 = 12$ network bits $+ 4$ subnet bits $= 20$ bits (`/20`).

3. **Subnetting Capacity:**
   * Borrowed bits ($n$) = 4 bits.
   * Possible subnets = $2^4 = 16$.
   * Usable host bits ($h$) = $32 - 20 = 12$ bits.
   * Usable hosts per subnet = $2^{12} - 2 = 4096 - 2 = 4094$.

4. **Determining the Specific Subnet for a Host:**
   * Apply a bitwise logical AND operation between the Host IP (`184.65.94.20`) and the Subnet Mask (`255.255.240.0`):

$$\begin{aligned}
\text{Host IP Third Octet (94):} \quad & 01011110_2 \\
\text{Mask Third Octet (240):} \quad & 11110000_2 \\
\hline
\text{Subnet ID Third Octet:} \quad & 01010000_2 = 80_{10}
\end{aligned}$$

* Resulting Subnet ID: `184.65.80.0/20`
* First Usable IP: `184.65.80.1`
* Usable Host Address Range: `184.65.80.1` to `184.65.95.254`
* Directed Broadcast Address: `184.65.95.255` (calculated by setting all 12 host bits to 1, which sets the last 4 bits of the third octet and all 8 bits of the fourth octet to 1: $80 + 15 = 95$).
