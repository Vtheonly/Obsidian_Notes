# 5.4 Morphological Algorithms

We can use the primitives ($ \oplus, \ominus, \circ, \bullet $) to extract structural information.

## 1. Boundary Extraction
How do we find the "outline" of a binary object $A$?
1.  Erode $A$ by a small $3 \times 3$ SE. This gives us a slightly smaller version of $A$.
2.  Subtract the eroded version from the original.
$$ \beta(A) = A - (A \ominus B) $$
The result is a 1-pixel thick boundary of the object.

## 2. Region Filling
If we have a boundary and want to fill the inside:
1.  Start with a point $X_0$ inside the boundary.
2.  Iteratively dilate the point: $X_k = (X_{k-1} \oplus B) \cap A^c$.
3.  The intersection with $A^c$ (the complement of the boundary) ensures the "growth" doesn't leak outside the boundary.
4.  Stop when $X_k = X_{k-1}$.

## 3. Skeletonization (Thinning)
The goal is to reduce an object to a 1-pixel wide "stick figure" that represents its topology.
*   **Concept:** Repeatedly erode the image but **do not** remove pixels if doing so would break the connectivity or remove an endpoint.
*   **Result:** A skeleton that is equidistant from the boundaries (the "medial axis").
