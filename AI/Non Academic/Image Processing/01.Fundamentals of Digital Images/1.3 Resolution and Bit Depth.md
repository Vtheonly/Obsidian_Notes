# 1.3 Resolution and Bit Depth

This note covers the two dimensions of quality in a digital image: "How many pixels?" (Spatial) and "How much information per pixel?" (Intensity).

## 1. Spatial Resolution
Spatial resolution is the measure of the smallest discernible detail in an image.

### A. Digital Dimensions (Pixel Count)
The most basic definition is the width and height of the grid.
*   Example: $1920 \times 1080$ (Full HD).
*   Total Pixels: $1920 \times 1080 = 2,073,600$ pixels ($\approx 2$ Megapixels).

### B. Physical Density (PPI and DPI)
This is often confused with pixel count. This metric links the digital pixels to the physical world.
*   **PPI (Pixels Per Inch):** Used for screens.
*   **DPI (Dots Per Inch):** Used for printers.

**Formula:**
$$ \text{Physical Size (Inches)} = \frac{\text{Pixel Dimension}}{\text{Resolution (PPI)}} $$

*   **Example:** An image is $1000$ pixels wide.
    *   Printed at $100$ DPI, it will be $10$ inches wide (Low quality print).
    *   Printed at $300$ DPI, it will be $3.33$ inches wide (High quality photo print).
*   **Insight:** Changing the DPI metadata in a file does **not** change the image data (the pixels); it only tells the printer how big to print it.

## 2. Intensity Resolution (Color Depth / Bit Depth)
Bit depth refers to the number of bits used to represent the color of a **single pixel**. This determines the **Dynamic Range** and the **Color Palette**.

### A. Calculation
If a pixel depth is $k$ bits, the number of unique colors/gray levels $L$ is:
$$ L = 2^k $$

### B. Common Depths
1.  **1-bit (Binary / Monochrome):**
    *   $2^1 = 2$ levels (0 and 1).
    *   Strictly Black and White. No gray. Used for text and line art.
2.  **8-bit (Grayscale):**
    *   $2^8 = 256$ levels (0 to 255).
    *   Standard for black and white photography.
    *   *Note:* The human eye can distinguish roughly 30-100 shades of gray depending on conditions, so 256 is usually sufficient to simulate continuity.
3.  **24-bit (True Color / RGB):**
    *   Uses 8 bits for Red, 8 bits for Green, 8 bits for Blue.
    *   Total: $2^8 \times 2^8 \times 2^8 = 16,777,216$ colors.
    *   This is the standard for JPEGs and monitors.
4.  **32-bit (RGBA):**
    *   24-bit color + 8-bit **Alpha Channel** (Transparency).
    *   Used in PNGs and game assets.

## 3. Storage Calculation
It is essential to be able to calculate the raw file size of an uncompressed image.

$$ \text{Size (bits)} = \text{Width} \times \text{Height} \times \text{Bit Depth} $$

To get Bytes, divide by 8.

**Exercise Example:**
Calculate the size of a standard Full HD ($1920 \times 1080$) image in True Color ($24$-bit).
1.  Total Pixels = $2,073,600$.
2.  Total Bits = $2,073,600 \times 24 = 49,766,400$ bits.
3.  Total Bytes = $49,766,400 / 8 = 6,220,800$ bytes.
4.  Convert to MB = $6.22$ MB.

*Tip:* This is why compression (JPEG) is necessary. Storing 6MB per photo is inefficient.
