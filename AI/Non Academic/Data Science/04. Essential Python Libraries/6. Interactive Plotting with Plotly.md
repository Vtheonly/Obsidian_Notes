# 6. Interactive Plotting with Plotly

**Plotly** is a graphing library that makes interactive, publication-quality graphs. Unlike Matplotlib (static images), Plotly renders HTML/JavaScript, allowing zooming, panning, and hovering.

## 6.1 Plotly Express (`px`)

`plotly.express` is the high-level API (similar to Seaborn) that covers 90% of use cases with minimal code.

```python
import plotly.express as px

# Scatter plot with hover data
fig = px.scatter(df, 
                 x='GDP', 
                 y='Life Expectancy', 
                 color='Continent',
                 hover_name='Country',
                 size='Population')
fig.show()
```

## 6.2 Key Features

1.  **Interactivity:** Users can hover over points to see exact values.
2.  **Zooming:** Selecting a region zooms into it.
3.  **Toggles:** Clicking legend items hides/shows groups.
4.  **Export:** Plots can be saved as interactive HTML files (`fig.write_html("plot.html")`).

## 6.3 Plotly Graph Objects (`go`)

For fine-grained control or creating complex dashboards with multiple types of charts overlaid, use `plotly.graph_objects`.

```python
import plotly.graph_objects as go

fig = go.Figure()
fig.add_trace(go.Scatter(x=x, y=y, mode='lines', name='Line'))
fig.add_trace(go.Bar(x=x, y=y2, name='Bar'))
fig.show()
