# 2. Pandas. Data Manipulation and Series

**Pandas** (Python Data Analysis Library) builds on top of NumPy to provide high-level data structures and analysis tools. It introduces the concept of **labels** (indices) to data, making it relational.

## 2.1 The Two Core Structures

1.  **Series:** A 1-dimensional labeled array (like a single column in Excel).
2.  **DataFrame:** A 2-dimensional labeled data structure (like a whole Excel sheet).

## 2.2 The Pandas Series

A Series holds data of any type (integers, strings, floating point numbers, Python objects, etc.).

### Creation

```python
import pandas as pd

# From List
s = pd.Series([10, 20, 30])

# From List with Custom Index
s = pd.Series([10, 20, 30], index=['a', 'b', 'c'])

# From Dictionary
data = {'a': 10, 'b': 20, 'c': 30}
s = pd.Series(data)
```

### Components

*   **Values:** The underlying NumPy array. Accessed via `s.values`.
*   **Index:** The labels. Accessed via `s.index`.

### Vectorized Operations (Alignment)

Pandas is powerful because it aligns data by **Index**, not by position.

```python
s1 = pd.Series([1, 2], index=['a', 'b'])
s2 = pd.Series([10, 20], index=['b', 'a']) # Note order flip

# Pandas matches 'a' to 'a' and 'b' to 'b'
print(s1 + s2)
# Output:
# a    21
# b    12
# dtype: int64
```

> [!WARNING] **Missing Data**
> If indices don't match during an operation, Pandas introduces `NaN` (Not a Number) to indicate missing data.
