# 5. Seaborn. Statistical Data Visualization

**Seaborn** is a high-level library built *on top* of Matplotlib. It integrates closely with Pandas DataFrames and provides beautiful default styles.

## 5.1 Philosophy

*   **Matplotlib:** "Draw a blue line from (0,0) to (1,1)." (Low level).
*   **Seaborn:** "Draw a scatterplot of Price vs. Size, colored by Neighborhood." (High level).

## 5.2 Key Plot Categories

Seaborn organizes plots into modules.

### 1. Relational Plots (Relationships)
*   `sns.scatterplot()`
*   `sns.lineplot()`
*   *Wrapper:* `sns.relplot()`

### 2. Distribution Plots (Distributions)
*   `sns.histplot()`: Histogram.
*   `sns.kdeplot()`: Kernel Density Estimate (Smooth curve).
*   `sns.boxplot()`: Summary statistics.
*   `sns.violinplot()`: Boxplot + KDE combined.
*   *Wrapper:* `sns.displot()`

### 3. Categorical Plots
*   `sns.barplot()`: Estimate of central tendency (mean).
*   `sns.countplot()`: Count of observations.
*   `sns.stripplot()`: Scatterplot for categorical data.

## 5.3 The Correlation Heatmap

One of the most useful tools in Data Science for Feature Selection.

```python
import seaborn as sns

# Calculate correlation matrix
corr = df.corr()

# Plot heatmap
sns.heatmap(corr, annot=True, cmap='coolwarm')
```

## 5.4 Pairplot

Visualizing pairwise relationships in an entire dataset.
```python
sns.pairplot(df, hue='Species')
```

> [!TIP] **Integration**
> Since Seaborn is built on Matplotlib, you can use Matplotlib commands to tweak Seaborn plots (e.g., `plt.title()`, `plt.figure()`).
