# 3. Outlier Detection and Treatment Methods

An **outlier** is a data point that differs significantly from other observations.
*   **Cause:** Measurement error, data entry error, or a genuine anomaly (e.g., Fraud).
*   **Impact:** Skews mean/std dev, destroys linear regression models.

## 3.1 Visual Detection

*   **Boxplot:** Points outside the whiskers are outliers.
    ```python
    sns.boxplot(x=df['Salary'])
    ```
*   **Scatterplot:** Points far away from the main cluster.

## 3.2 Statistical Method: Z-Score

Measures how many standard deviations a point is from the mean.
*   **Rule:** If $|Z| > 3$, it is an outlier.
*   **Assumption:** Data is Normally Distributed (Bell Curve).

```python
from scipy import stats
import numpy as np

z_scores = np.abs(stats.zscore(df['Salary']))
# Filter out rows with Z-score > 3
df_clean = df[(z_scores < 3)]
```

## 3.3 Statistical Method: IQR (Interquartile Range)

Robust to non-normal distributions. Used in Boxplots.

1.  Calculate **Q1** (25th percentile) and **Q3** (75th percentile).
2.  Calculate **IQR = Q3 - Q1**.
3.  Define bounds:
    *   Lower Bound = $Q1 - 1.5 \times IQR$
    *   Upper Bound = $Q3 + 1.5 \times IQR$
4.  Any value outside these bounds is an outlier.

```python
Q1 = df['Age'].quantile(0.25)
Q3 = df['Age'].quantile(0.75)
IQR = Q3 - Q1

# Filter
df_clean = df[~((df['Age'] < (Q1 - 1.5 * IQR)) | (df['Age'] > (Q3 + 1.5 * IQR)))]
```

## 3.4 Treatment Strategies

1.  **Drop:** If it's an error or irrelevant.
2.  **Cap (Winsorization):** Replace values > 99th percentile with the 99th percentile value.
    ```python
    upper_limit = df['Age'].quantile(0.99)
    df.loc[df['Age'] > upper_limit, 'Age'] = upper_limit
    ```
3.  **Transform:** Apply Log or Square Root to compress the range (reduces impact of large values).
    ```python
    df['Log_Income'] = np.log(df['Income'])
