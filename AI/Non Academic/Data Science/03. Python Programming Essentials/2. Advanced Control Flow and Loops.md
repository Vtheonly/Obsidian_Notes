# 2. Advanced Control Flow and Loops

Control flow is the order in which individual statements, instructions, or function calls are executed or evaluated.

## 2.1 Conditional Logic

The `if...elif...else` structure allows the program to make decisions.

```python
grade = 85

if grade >= 90:
    print("A")
elif grade >= 80:
    print("B")
else:
    print("Fail")
```

**Indentation is critical.** Python uses whitespace (4 spaces) to define the scope of the block.

## 2.2 Loops

### The `for` Loop
Used for iterating over a sequence (list, tuple, dictionary, set, or string).

```python
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)
```

**The `range()` function:**
Generates a sequence of numbers. Syntax: `range(start, stop, step)`.
*   `range(5)` -> 0, 1, 2, 3, 4 (Stop is exclusive).
*   `range(1, 10, 2)` -> 1, 3, 5, 7, 9.

### The `while` Loop
Executes a set of statements as long as a condition is true.

```python
count = 0
while count < 5:
    print(count)
    count += 1  # Crucial: Increment to avoid infinite loops
```

## 2.3 Loop Controls

*   **break:** Terminates the loop immediately.
*   **continue:** Skips the rest of the current iteration and jumps to the next one.
*   **pass:** A null operation. Nothing happens. Used as a placeholder for future code.

## 2.4 List Comprehensions (Advanced)

A concise way to create lists. It is faster and more Pythonic than standard loops.

**Syntax:** `[expression for item in iterable if condition]`

**Example:**
Create a list of squares for even numbers only.

*   **The Old Way:**
    ```python
    squares = []
    for x in range(10):
        if x % 2 == 0:
            squares.append(x**2)
    ```

*   **The Pythonic Way:**
    ```python
    squares = [x**2 for x in range(10) if x % 2 == 0]
    ```

> [!TIP] **Performance**
> List comprehensions are generally faster than `for` loops because the iteration is performed at C-language speed inside the interpreter.
