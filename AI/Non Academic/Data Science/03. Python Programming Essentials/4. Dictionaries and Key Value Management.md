# 4. Dictionaries and Key Value Management

## 4.1 Definition

A **Dictionary** is an unordered, mutable collection of key-value pairs.
*   **Keys:** Must be unique and immutable (strings, numbers, tuples).
*   **Values:** Can be any object.
*   **Syntax:** `my_dict = {"name": "Alice", "age": 25}`

## 4.2 Accessing Data

```python
student = {
    "name": "John",
    "grades": [85, 90, 88]
}

# Standard Access
print(student["name"])  # "John"

# Error Handling
# print(student["gender"]) -> Raises KeyError because key doesn't exist
```

### The `.get()` Method
Safely accesses a key. If the key is missing, it returns `None` (or a default value) instead of crashing.
```python
print(student.get("gender", "Unknown"))  # Returns "Unknown"
```

## 4.3 Modifying Dictionaries

*   **Add/Update:** `student["age"] = 22`
*   **Delete:** `del student["grades"]`

## 4.4 Iteration

When looping through a dictionary, you have three options:

1.  **Keys (Default):**
    ```python
    for k in student:
        print(k)
    ```
2.  **Values:**
    ```python
    for v in student.values():
        print(v)
    ```
3.  **Items (Both):**
    ```python
    for k, v in student.items():
        print(f"{k}: {v}")
    ```

## 4.5 Dictionary Comprehensions

Similar to list comprehensions, but for creating dictionaries.

```python
keys = ['a', 'b', 'c']
values = [1, 2, 3]

# Create dict mapping key to value
my_dict = {k: v for k, v in zip(keys, values)}
# Result: {'a': 1, 'b': 2, 'c': 3}
