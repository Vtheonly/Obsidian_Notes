## 8.3 Model Artifacts (.h5, .pkl, .json)
To use the model without retraining it, we export the "Brain" into artifacts:
*   **`.h5` / `.pt` files:** These store the architecture and millions of optimized mathematical weights of the Neural Network (Keras/PyTorch).
*   **`.json` files:** Often used to store the specific tree structures and split thresholds of the XGBoost model.
*   **`.pkl` (Pickle) files:** These store the exact `RobustScaler` object. This is critical: if the model learned that $1500$ kW equals $0.8$ internally, the exact same scaler must be loaded in production to translate the data identically.
