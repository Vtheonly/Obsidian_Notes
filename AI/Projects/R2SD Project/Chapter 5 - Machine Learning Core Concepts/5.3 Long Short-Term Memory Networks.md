## 5.3 Long Short-Term Memory (LSTM) Networks
LSTMs are a special type of RNN specifically designed to solve the vanishing gradient problem and learn long-term dependencies.

They achieve this using a **Cell State ($C_t$)**—a continuous "conveyor belt" of memory that runs straight down the entire sequence. This cell state is protected by three specialized **Gates** (which are essentially small neural networks with sigmoid activation functions, outputting values between 0 and 1):
1.  **Forget Gate ($f_t$):** Decides what information from the previous cell state ($C_{t-1}$) should be *discarded*. If the weather was clear 10 hours ago but a storm just started, the forget gate might "forget" the clear sky memory.
2.  **Input Gate ($i_t$):** Decides what *new* information from the current input and previous hidden state should be *stored* in the cell state.
3.  **Output Gate ($o_t$):** Controls what parts of the cell state should be outputted as the hidden state ($h_t$) for the current time step.

These gates allow LSTMs to selectively remember or forget information over long periods, making them ideal for complex time-series data like weather.
