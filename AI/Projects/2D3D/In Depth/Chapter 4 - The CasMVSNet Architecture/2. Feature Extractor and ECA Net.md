# 2. Feature Extractor and ECA Net

Before building the 3D volume, the network extracts 2D features from the images. 
It uses an 8-layer Convolutional Neural Network (CNN) backbone. 

## Shared Weights
The exact same CNN is applied independently to both the Reference image and the Source images. The goal is to translate raw RGB pixels into high-dimensional semantic features.

## ECA-Net (Efficient Channel Attention)
Deep CNNs frequently cause thin, sharp structures (like scaffolding or roof edges) to mathematically blur into the background. 

The architecture injects **ECA Blocks** after the 3rd and 6th convolutions. 
ECA works by:
1.  Applying Adaptive Average Pooling to compress the spatial dimensions into a $1 \times 1$ block.
2.  Running a 1D Convolution across the channels to discover which feature maps are currently tracking important edges.
3.  Applying a Sigmoid function to generate an attention "weight" mask.
4.  Multiplying the original features by this mask. 

This explicitly forces the network to prioritize sharp architectural lines, preventing windows and corners from melting.
