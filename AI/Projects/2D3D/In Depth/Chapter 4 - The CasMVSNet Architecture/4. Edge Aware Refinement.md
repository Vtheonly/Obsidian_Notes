```markdown
# 4. Edge Aware Refinement

Even with Soft Argmin, Deep Learning MVS tends to "oversmooth" sharp object boundaries. If a building edge meets the infinite sky, the depth map often creates a "melted" ramp between the building and the sky instead of a sharp drop-off.

To fix this, CasMVSNet adds a final **Edge-Aware Refinement Network** (approx ~0.5M parameters).

1.  **Input:** The final depth prediction from Stage 3 ($D_3$, 1 channel) is concatenated with the raw, original RGB reference image (3 channels) forming a 4-channel tensor.
2.  **Architecture:** Passed through a lightweight 2D CNN mapping 96 channels across 3 ResNet blocks.
3.  **Function:** The network uses the sharp color contrast in the RGB image (e.g., dark roof against bright blue sky) to mathematically "snap" the blurry depth gradient to the exact physical boundary.
4.  **Result:** The residual corrections are added back to the coarse depth, yielding perfectly sharp metric edges primed for TSDF Fusion.
```
