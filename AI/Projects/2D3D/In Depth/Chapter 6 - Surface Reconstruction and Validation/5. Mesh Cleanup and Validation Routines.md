# 5. Mesh Cleanup and Validation Routines

A raw mesh generated by Marching Cubes or Poisson reconstruction often contains geometric artifacts that can cause problems for rendering, simulation, or 3D printing. The `clean_mesh` function in your `mesh_postprocess.py` module is a critical sanitization step.

## Key Cleanup Operations
Your code calls a sequence of specific Open3D functions. Here is what each one does and why it's necessary:

*   **`remove_degenerate_triangles()`**: A degenerate triangle has an area of exactly `0.0`. This happens when all three of its vertices lie on a single line. These triangles are invisible but add useless data to the mesh file and can crash some rendering engines. This function finds and removes them.

*   **`remove_duplicated_triangles()`**: Sometimes, the meshing algorithm can accidentally create the exact same triangle (using the same three vertex indices) twice. This adds no new information and bloats the file size. This function identifies and removes these redundant faces.

*   **`remove_duplicated_vertices()`**: Due to floating-point inaccuracies, two vertices that should be in the same location might be created with infinitesimally different coordinates (e.g., `(1.0, 2.0, 3.0)` and `(1.0000000001, 2.0, 3.0)`). This function merges vertices that are extremely close together into a single vertex, "welding" the mesh and reducing its complexity.

*   **`remove_unreferenced_vertices()`**: After removing degenerate or duplicated triangles, some vertices may no longer be part of any triangle. They become "orphan" points floating in space. This function scans the mesh and deletes any vertex that is not referenced by at least one triangle face.

## Watertight Validation (`is_watertight()`)
A "watertight" or "manifold" mesh is one that represents a fully enclosed solid with no holes. The `is_watertight()` check in your `validate_output.py` script performs an important test: it checks that **every edge in the mesh is shared by exactly two triangles**.

*   If an edge is only used by one triangle, it's a hole or a boundary.
*   If an edge is used by three or more triangles, it represents non-manifold geometry (e.g., two cubes joined at an edge), which is physically impossible and will break many algorithms.

For partial scans (like reconstructing only the front of a building), a non-watertight mesh is expected and perfectly normal.
