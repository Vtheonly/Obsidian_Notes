# 3. Differentiable Homography Warping

In advanced Neural MVS architectures (like CasMVSNet), the network must be taught how to do stereopsis. It does this using **Homography Warping**.

Homography is a transformation that maps a flat plane from one camera view to another. Because we evaluate depth by "sweeping" planes across the scene, we can use homography to mathematically warp the pixels of a Source Image into the perspective of the Reference Image.

## The Mathematics
Given the Intrinsic matrix $K$, the Rotation $R$, Translation $t$, and a hypothetical depth $d$, the Homography Matrix $H$ mapping the Source to the Reference is:

$$H_{src \rightarrow ref} = K \cdot \left( R + \frac{t \cdot n^T}{d} \right) \cdot K^{-1}$$

*(Where $n^T$ is the plane normal vector, usually pointing straight down the optical axis: $[0, 0, 1]$).*

By applying this warp inside PyTorch (`F.grid_sample(mode='bilinear')`), the neural network can directly "look" at how well the pixels align at different depth hypotheses, allowing it to learn Multi-View Stereo dynamically.
