# 4. SIFT Matching and The Lowe's Ratio Test

Your `FeatureMatcher` class is the practical implementation of the first steps of SfM. It relies on two key components from OpenCV: SIFT and a Brute-Force Matcher with a specific filtering technique.

## SIFT (Scale-Invariant Feature Transform)
SIFT is an algorithm that finds "keypoints" in an image that are robust to changes in scale, rotation, and illumination.
1.  **Detection:** It finds points of high contrast or corners that are unique.
2.  **Descriptor:** For each keypoint, it creates a 128-dimensional vector (a "fingerprint") that describes the local image gradients around it. This fingerprint is what allows for matching.

## Brute-Force Matching (`cv2.BFMatcher`)
The `BFMatcher` is the simplest matching strategy. It takes the 128-d descriptor for one keypoint in Image A and compares its Euclidean distance to *every single descriptor* in Image B, finding the one with the smallest distance.

## Lowe's Ratio Test
This "brute-force" approach can lead to many false positives. The `ratio_threshold` parameter in your code implements a crucial filtering step known as Lowe's Ratio Test.

```python
raw_matches = self.bf.knnMatch(des1, des2, k=2)
# ...
if m.distance < self.ratio_threshold * n.distance:
    good.append(m)
```
Instead of asking for just the single best match (`k=1`), we ask for the **two best matches** (`k=2`). Let's call them `m` (the best) and `n` (the second-best).

*   **The Logic:** For a match to be considered truly reliable and unambiguous, the best match should be *significantly better* than the second-best match. If the best and second-best matches have very similar distances, it means the keypoint is not very unique (e.g., a repeating brick pattern), and the match is likely incorrect.
*   **The Threshold:** A `ratio_threshold` of `0.75` (a standard value) means we only accept a match if the distance to the best match is less than 75% of the distance to the second-best match. This simple ratio test is extremely effective at filtering out ambiguous and incorrect feature correspondences.
