# 2. Point Cloud Primitives from Scratch

The `_PointCloud` class in `o3d_compat.py` demonstrates how core 3D operations can be built using only NumPy array manipulations.

## Voxel Down-sampling in Pure NumPy
The `voxel_down_sample` method is a key optimization for handling dense point clouds. Here's how the NumPy version works:

```python
q = np.floor(self._points / voxel_size).astype(np.int64)
_, ui = np.unique(q, axis=0, return_index=True)
o._points = self._points[ui].copy()
```
1.  **Quantization:** Each point's $(X, Y, Z)$ coordinate is divided by the `voxel_size` and floored. This effectively assigns every point to an integer grid coordinate. All points that fall within the same 3D "bin" or "voxel" will now have the same integer coordinate.
2.  **`np.unique`:** This is the core of the algorithm. It looks at the array of quantized coordinates (`q`) and finds all the unique rows.
3.  **`return_index=True`:** This is the magic. Along with the unique rows, it returns the index (`ui`) of the *first occurrence* of each unique row.
4.  **Selection:** By indexing the original `_points` array with these unique indices (`_points[ui]`), we are effectively selecting just one representative point from each occupied voxel. This achieves a massive and efficient down-sampling of the point cloud.

## Normal Estimation from Scratch
The `estimate_normals` function is another brilliant piece of from-scratch engineering to avoid SciPy's `KDTree`.

1.  **Chunked Brute-Force:** To avoid memory overload, it processes points in chunks. For each point `q` in a chunk, it calculates the squared distance to *every other point* in the full point cloud.
2.  **`np.argpartition`:** This is a highly optimized NumPy function that is faster than a full sort. `np.argpartition(d, k)` finds the indices of the `k` smallest elements without sorting them completely. This is used to efficiently find the `k`-nearest neighbors (KNN).
3.  **PCA on Neighbors:** For the `k` neighbors of a point, it computes their covariance matrix (`cov = c.T @ c`). It then finds the eigenvectors of this matrix. The eigenvector corresponding to the *smallest* eigenvalue represents the direction with the least variance, which is, by definition, the normal of the plane that best fits the `k` points. This is a classic application of Principal Component Analysis (PCA) to find surface normals.
