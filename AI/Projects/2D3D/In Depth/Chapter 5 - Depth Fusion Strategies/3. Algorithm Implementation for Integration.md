# 3. Algorithm Implementation for Integration

How do you physically program TSDF Integration into a voxel grid? The underlying math is highly specific. 

To merge continuous space, we track a running average. Every voxel $x$ tracks two values: an accumulated weighted distance `Sum_WD` and an accumulated weight `Sum_W`.

## Step-by-Step Logic
For a given camera view $i$:
1.  **Iterate Voxels:** Loop over every 3D coordinate representing a voxel center $(X, Y, Z)$.
2.  **Project:** Use the Intrinsic $K$ and Extrinsic $[R|t]$ matrices to project the voxel into 2D pixel coordinates $(u, v)$ in the current image.
3.  **Extract Depth:** Read the depth value actually measured by the camera at that pixel: $d_{measured} = DepthImage(u, v)$.
4.  **Calculate Voxel Depth:** Measure the physical distance from the camera center to the voxel center: $d_{voxel}$.
5.  **Compute SDF:** Calculate the signed distance $dist = d_{measured} - d_{voxel}$. 
    * If $dist > 0$, the voxel is in front of the surface.
    * If $dist < 0$, the voxel is behind it.
6.  **Calculate Weight:** Apply $W = \frac{1}{d_{measured}^2}$. This is critical. A pixel measured 1 meter away is exponentially more reliable than a pixel measured 10 meters away.
7.  **Accumulate:** 
    * `Sum_WD += W * dist`
    * `Sum_W += W`

When all cameras are finished, you determine the final value of the voxel by dividing: `Value = Sum_WD / Sum_W`. If the value passes exactly through $0.0$, a surface polygon is generated.
