# 2.4 Histogram Equalization (HE)

While contrast stretching spreads the histogram linearly, **Histogram Equalization** transforms the image so that the output histogram is approximately **uniform** (flat).

## 1. The Goal: Maximizing Entropy
Information theory tells us that an image contains the most information (maximum entropy) when all gray levels are equiprobable.
*   Visually, this means the image has optimal contrast across all gray levels.
*   We want a transformation $s = T(r)$ such that $p_s(s) = \text{constant}$.

## 2. The Mathematical Foundation (CDF)
The transformation function that accomplishes this is the **Cumulative Distribution Function (CDF)** of the input histogram.

$$ s_k = T(r_k) = (L-1) \sum_{j=0}^{k} p_r(r_j) $$

Where:
*   $L$ is the number of gray levels (256).
*   $p_r(r_j)$ is the probability (normalized count) of gray level $j$.
*   The sum $\sum p_r$ is the Cumulative Probability.

### Why the CDF?
1.  **Monotonicity:** The CDF is monotonically increasing ($T(r)$ never goes down as $r$ goes up). This preserves the order of brightness (dark pixels stay darker than bright pixels).
2.  **Range:** The CDF outputs values between 0 and 1. Multiplying by $(L-1)$ scales it to the image range.

## 3. Discrete Example Step-by-Step
Assume a 3-bit image ($L=8$, values 0-7) with total pixels $N$.

1.  **Count Pixels ($n_k$):**
    *   $r_0=0$: 10 pixels
    *   $r_1=1$: 20 pixels ... etc.
2.  **Calculate PDF ($p_r$):** Divide counts by $N$.
3.  **Calculate CDF:** Accumulate the probabilities.
    *   $CDF_0 = p_0$
    *   $CDF_1 = p_0 + p_1$
    *   $CDF_2 = p_0 + p_1 + p_2$
4.  **Map New Levels ($s_k$):**
    *   $s_k = \text{round}((L-1) \times CDF_k)$
    *   $s_k = \text{round}(7 \times CDF_k)$
5.  **Replace:** Go through the original image. Every pixel with value $r_0$ is replaced by $s_0$. Every $r_1$ by $s_1$, etc.

## 4. Visual Result
*   The histogram is flattened.
*   The image typically shows significantly increased contrast.
*   **Artifacts:** Because discrete values are grouped together, the histogram won't be perfectly flat; it will look "sparse" or comb-like, but the cumulative distribution will be linear.
