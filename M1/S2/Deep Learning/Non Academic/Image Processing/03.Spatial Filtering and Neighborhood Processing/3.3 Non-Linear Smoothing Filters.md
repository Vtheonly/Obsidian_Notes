# 3.3 Non-Linear Smoothing Filters (Rank-Order)

Non-linear filters do not use a weighted sum. Instead, they **sort** the pixels in the neighborhood and pick a value based on their rank.

## 1. The Median Filter
The most powerful tool for removing **Impulse Noise (Salt & Pepper)**.

### How it works
1.  Select a neighborhood (e.g., $3 \times 3$).
2.  List the 9 pixel values.
3.  **Sort** them in ascending order.
4.  Pick the middle value (the 5th value for $3 \times 3$).
5.  Assign this value to the center pixel.

### Why it is better than Mean for Noise
*   **The Mean Filter** is sensitive to outliers. If you have 8 black pixels (0) and 1 "salt" noise pixel (255), the mean is 28. The noise survives as a gray smudge.
*   **The Median Filter** ignores outliers. In the same scenario, the sorted list is `[0,0,0,0,0,0,0,0,255]`. The median is 0. The noise is **completely deleted**.

## 2. Min and Max Filters
*   **Max Filter:** Picks the largest value in the neighborhood.
    *   *Use:* Removes "Pepper" noise (black dots) and brightens the image.
*   **Min Filter:** Picks the smallest value.
    *   *Use:* Removes "Salt" noise (white dots) and darkens the image.

## 3. Comparison Table

| Feature | Mean Filter | Median Filter |
| :--- | :--- | :--- |
| **Type** | Linear (Weighted Sum) | Non-Linear (Order Statistics) |
| **Best For** | Gaussian (Uniform) Noise | Salt & Pepper (Impulsive) Noise |
| **Edge Quality** | Blurs edges significantly | Preserves sharp edges well |
| **Computation** | Fast (Simple arithmetic) | Slower (Requires sorting) |
