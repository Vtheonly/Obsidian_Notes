# 5.1 Mathematical Morphology Basics

## 1. Introduction
Mathematical morphology is a theory and technique for the analysis and processing of geometrical structures, based on **set theory**, lattice theory, and topology. 

While spatial filters (Chapter 3) are based on arithmetic operations (sums, products), morphological operations are based on **geometry**. We probe an image with a small shape called a **Structuring Element (SE)** and observe how it fits or misses the shapes in the image.

## 2. Set Theory Foundations
In morphology, an image is treated as a set of coordinates.
*   **Binary Images:** A set in $\mathbb{Z}^2$. If a pixel at $(x,y)$ is 1 (white), it belongs to the set $A$. If it is 0 (black), it does not.
*   **Foreground vs. Background:** By convention, morphology is performed on the foreground (white pixels).

## 3. The Structuring Element (SE)
The Structuring Element is the "kernel" of morphology. It is a small set used to probe the input image.

### A. Attributes of an SE
1.  **Shape:** Can be a square, disk, cross, or line. The shape you choose determines which features are preserved. (e.g., a horizontal line SE will "delete" vertical lines).
2.  **Origin:** One pixel in the SE is designated as the origin (usually the center). This is the "reference point" that corresponds to the pixel currently being processed in the input image.
3.  **Size:** A $3 \times 3$ SE has a small effect; a $21 \times 21$ SE has a massive effect.

### B. Reflection and Translation
*   **Reflection ($\hat{B}$):** The set of points in $B$ rotated $180^\circ$ around the origin.
*   **Translation ($B_z$):** Shifting the entire SE so its origin is at coordinate $z$.

## 4. The "Probe" Philosophy
Think of the SE as a physical probe or a "cookie cutter."
*   If we slide it over the image, we can ask:
    *   "Does the SE **fit** entirely inside the object?"
    *   "Does the SE **touch** the object at any point?"
