# 1. Variables, Operators, and Type Casting

## 1.1 Variables and Dynamic Typing

In Python, a variable is a symbolic name that is a reference or pointer to an object. Once an object is assigned to a variable, you can refer to the object by that name.

Unlike statically typed languages (like C++ or Java), Python is **dynamically typed**. You do not declare the type of the variable; the interpreter infers it at runtime based on the value assigned.

```python
x = 5           # x is an integer
x = "Hello"     # x is now a string (Reassignment is allowed)
```

### Naming Conventions (PEP 8)
*   **Variables/Functions:** `snake_case` (e.g., `student_age`, `calculate_mean`).
*   **Constants:** `UPPER_CASE` (e.g., `MAX_ITERATIONS`, `PI`).
*   **Classes:** `PascalCase` (e.g., `LinearRegression`, `DataFrame`).

## 1.2 Standard Data Types

| Type | Name | Example | Description |
| :--- | :--- | :--- | :--- |
| **int** | Integer | `42`, `-5` | Whole numbers without decimals. |
| **float** | Floating Point | `3.14`, `1.0` | Numbers with decimals. Note: `1` is an int, `1.0` is a float. |
| **str** | String | `"Python"`, `'Data'` | Sequence of characters. Immutable. |
| **bool** | Boolean | `True`, `False` | Logical truth values. Capitalization matters. |
| **NoneType** | None | `None` | Represents the absence of a value (null). |

## 1.3 Operators

### Arithmetic Operators
*   `+`, `-`, `*`: Standard addition, subtraction, multiplication.
*   `/`: **True Division**. Always returns a float. (`10 / 2` -> `5.0`).
*   `//`: **Floor Division**. Truncates the decimal part. (`10 // 3` -> `3`).
*   `%`: **Modulus**. Returns the remainder. (`10 % 3` -> `1`). Useful for checking even/odd numbers.
*   `**`: **Exponentiation**. Power. (`2 ** 3` -> `8`).

### Comparison Operators
Return a `bool` (`True` or `False`).
*   `==`: Equal to.
*   `!=`: Not equal to.
*   `>`, `<`, `>=`, `<=`: Greater/Less than logic.

### Logical Operators
Used to combine conditional statements.
*   `and`: Returns True if both statements are true.
*   `or`: Returns True if at least one statement is true.
*   `not`: Reverses the result.

### Membership Operators
*   `in`: Returns True if a sequence with the specified value is present in the object.
    ```python
    x = ["apple", "banana"]
    print("banana" in x) # True
    ```

## 1.4 Type Casting (Conversion)

Converting a variable from one data type to another.

### Implicit Conversion
Python automatically converts types when it makes sense without losing data.
```python
x = 10      # int
y = 2.5     # float
z = x + y   # z becomes 12.5 (float) automatically
```

### Explicit Conversion
Manually forcing a type change.
1.  **int()**: Truncates floats. `int(3.9)` -> `3`.
2.  **float()**: Adds decimal. `float(3)` -> `3.0`.
3.  **str()**: Converts any object to string representation. `str(100)` -> `"100"`.
4.  **bool()**: Returns `False` for 0, empty containers, or None. Returns `True` for everything else.

> [!WARNING] **Data Loss**
> Converting a string to an int (`int("hello")`) will raise a `ValueError`.
