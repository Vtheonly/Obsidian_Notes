# 4. Matplotlib. Static Data Visualization

**Matplotlib** is the foundational plotting library for Python. It provides complete control over every pixel of a figure.

## 4.1 Pyplot vs. Object-Oriented (OO) API

There are two ways to use Matplotlib. Mixing them causes confusion.

### 1. The Pyplot Interface (State-based)
Good for quick, simple interactive plots.
```python
import matplotlib.pyplot as plt
plt.plot([1, 2, 3], [1, 4, 9])
plt.title("Simple Plot")
plt.show()
```

### 2. The Object-Oriented Interface (Recommended)
Good for complex scripts, dashboards, and complete control.
```python
fig, ax = plt.subplots()  # Create Figure and Axes objects
ax.plot([1, 2, 3], [1, 4, 9])
ax.set_title("OO Plot")
ax.set_xlabel("X Axis")
plt.show()
```

## 4.2 Anatomy of a Figure

*   **Figure (`fig`):** The top-level container (the canvas). It holds everything.
*   **Axes (`ax`):** The region where data is plotted (a graph). A Figure can contain multiple Axes (subplots).
*   **Axis:** The x-axis and y-axis lines (spines), ticks, and labels.

## 4.3 Common Plot Types

*   `ax.plot(x, y)`: Line plot.
*   `ax.scatter(x, y)`: Scatter plot.
*   `ax.bar(x, height)`: Bar chart.
*   `ax.hist(data, bins)`: Histogram (distribution).
*   `ax.boxplot(data)`: Box and Whisker plot.

## 4.4 Customization

```python
ax.plot(x, y, color='red', linestyle='--', marker='o', label='Trend')
ax.legend()       # Show legend
ax.grid(True)     # Show grid
plt.savefig('my_plot.png') # Save to disk
