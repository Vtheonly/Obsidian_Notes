# 1. Handling Missing Data and Imputation Strategies

Real-world data is messy. One of the most common issues is **missing values** (NaN, null, None).
*   **Impact:** Most Machine Learning algorithms (like Scikit-Learn's Linear Regression) cannot handle missing values and will crash.
*   **Goal:** Replace missing data with statistically sound estimates or remove it entirely without biasing the dataset.

## 1.1 Identifying Missing Data

Before fixing it, you must find it.

```python
import pandas as pd
import seaborn as sns

df = pd.read_csv('data.csv')

# 1. Check for nulls (Boolean mask)
print(df.isnull())

# 2. Count missing values per column
print(df.isnull().sum())

# 3. Visual Heatmap of missingness
sns.heatmap(df.isnull(), cbar=False)
```

## 1.2 Types of Missing Data

Understanding *why* data is missing dictates *how* to fix it.

1.  **MCAR (Missing Completely at Random):** No pattern. (e.g., A sensor battery died randomly).
    *   *Safe to drop rows.*
2.  **MAR (Missing at Random):** Missingness depends on *observed* data. (e.g., Men are less likely to report their depression score, but gender *is* recorded).
    *   *Imputation is preferred.*
3.  **MNAR (Missing Not at Random):** Missingness depends on the *unobserved* value itself. (e.g., People with very high incomes skip the "Income" question).
    *   *Dangerous. Requires specialized modeling.*

## 1.3 Strategy 1: Deletion

Best when missing data is MCAR and less than 5% of total.

*   **Listwise Deletion (Rows):**
    ```python
    df.dropna(axis=0, inplace=True)
    ```
*   **Pairwise Deletion (Cols):**
    ```python
    df.dropna(axis=1, inplace=True) # Drop column if any value is missing
    df.dropna(thresh=5)             # Drop row if it has < 5 non-null values
    ```

## 1.4 Strategy 2: Simple Imputation

Replacing missing values with a summary statistic.

*   **Mean:** Good for normally distributed data.
    ```python
    df['Age'].fillna(df['Age'].mean(), inplace=True)
    ```
*   **Median:** Robust to outliers (Best general choice).
    ```python
    df['Salary'].fillna(df['Salary'].median(), inplace=True)
    ```
*   **Mode:** For categorical data.
    ```python
    df['City'].fillna(df['City'].mode()[0], inplace=True)
    ```
*   **Constant:** Filling with specific value (e.g., 0 or "Unknown").

## 1.5 Strategy 3: Advanced Imputation

Using algorithms to predict the missing value based on other features.

*   **KNN Imputer:** Finds 'k' most similar rows and averages their values.
    ```python
    from sklearn.impute import KNNImputer
    imputer = KNNImputer(n_neighbors=5)
    df_filled = imputer.fit_transform(df)
    ```
*   **Iterative Imputer (MICE):** Models each feature with missing values as a function of other features.
*   **Time-Series Interpolation:** Filling gaps based on time trends.
    ```python
    df['Sales'].interpolate(method='linear', inplace=True)
