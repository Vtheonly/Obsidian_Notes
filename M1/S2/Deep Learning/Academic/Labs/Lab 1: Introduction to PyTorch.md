## Setup

To install PyTorch locally (as noted in the slides), you must align your installation with your hardware.
If using an NVIDIA GPU, ensure CUDA is installed.

```bash
# Example installation for CUDA 11.8
pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```

## Two Major Frameworks

1. **PyTorch (Meta/Facebook):** Highly flexible, "pythonic", and the absolute standard for AI Research.
2. **TensorFlow (Google):** Robuster for large-scale industrial deployment, though PyTorch is rapidly catching up in production environments.
