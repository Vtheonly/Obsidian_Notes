# 9.1. The Perceptron & Complete Activation Functions

The Perceptron is the simplest neural network: a single layer with a single neuron. It separates linearly separable data.

### Two-Step Process

1.  **Aggregation:** $z = \sum w_j x_j + b$
2.  **Activation:** $a = f(z)$

### The Crucial Role of Non-Linearity

Without activation functions, stacking 100 hidden layers is mathematically identical to having 1 hidden layer. The network would only be capable of linear transformations. **Non-linear activation functions multiply the geometric complexity of the model**, allowing it to warp a flat plane into a highly complex sculpted surface capable of separating intricate data patterns.

### Exhaustive List of Activation Functions

1.  **Sigmoid:** $\sigma(x) = \frac{1}{1 + e^{-x}}$
    - _Usage:_ Binary Classification Output Layer.
    - _Problem:_ Vanishing gradients in deep networks.
2.  **Tanh (Hyperbolic Tangent):** $\tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$
    - _Usage:_ Hidden layers (historically). Similiar to Sigmoid but centered at 0 (range -1 to 1), which facilitates faster convergence.
3.  **Softmax:** $Softmax(z_i) = \frac{e^{z_i}}{\sum_{j} e^{z_j}}$
    - _Usage:_ Multi-class Classification Output Layer. Transforms scores into a valid probability distribution (all outputs sum to 1).
4.  **ReLU (Rectified Linear Unit):** $f(x) = \max(0, x)$
    - _Usage:_ **The Standard for Hidden Layers.** Extremely fast to calculate. Solves the vanishing gradient problem.
5.  **Leaky ReLU:** $f(x) = \max(\alpha x, x)$ where $\alpha$ is a small constant (e.g., 0.01).
    - _Usage:_ Fixes the "Dying ReLU" problem (where neurons output 0 permanently and "die" because the gradient is 0).
6.  **PReLU (Parametric ReLU):** Same as Leaky ReLU, but $\alpha$ is not a fixed constant; it is a parameter learned by the network during training.
7.  **ELU (Exponential Linear Unit):** $f(x) = x$ if $x > 0$, else $\alpha(e^x - 1)$.
    - _Usage:_ Tends to converge faster and produces activations closer to zero.
8.  **Swish:** $f(x) = x \cdot \sigma(x)$
    - _Usage:_ Discovered by Google. Often outperforms ReLU in extremely deep networks (40+ layers).
9.  **GELU (Gaussian Error Linear Unit):** $f(x) = x \cdot \Phi(x)$
    - _Usage:_ Massively used in **Transformer architectures** (BERT, GPT) due to its probabilistic nature.
