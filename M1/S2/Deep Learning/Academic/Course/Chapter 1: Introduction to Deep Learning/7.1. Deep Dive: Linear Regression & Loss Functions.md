# 7.1. Deep Dive: Linear Regression & Loss Functions

Linear regression is a supervised predictive method. We are establishing a line of best fit to determine the relationship (correlation/causality) between an Exogenous variable $X$ (independent) and an Endogenous variable $Y$ (dependent).

### The Hypothesis Function

$$ h(x) = \theta_0 + \theta_1 x $$

- $\theta_0$: The y-intercept (Bias).
- $\theta_1$: The slope (Weight).
- In a multi-variable scenario: $h(x) = \theta_0 + \theta_1 x_1 + \theta_2 x_2 ... \theta_n x_n$.
- Vectorized form: $h(x) = \theta^T x$ (where $x_0 = 1$).

### The Cost Function (MSE)

We use the Method of Least Squares, formulated as the Mean Squared Error (MSE).

$$ J(\theta*0, \theta_1) = \frac{1}{2m} \sum*{i=1}^{m} (h\_\theta(x^{(i)}) - y^{(i)})^2 $$

> [!WARNING] The 1/2m Trick
> Notice the $\frac{1}{2m}$ instead of just $\frac{1}{m}$? This is a deliberate mathematical convenience. When we calculate the derivative of this function (to find the gradient), the exponent $2$ from the square drops down. The $2$ cancels out the $1/2$, leaving a very clean, simple equation without lingering constants.

### Applying Gradient Descent

To minimize $J(\theta)$, we calculate the partial derivatives with respect to $\theta_0$ and $\theta_1$:

1.  **Derivative for $\theta_0$ ($j=0$):**
    $$ \frac{\partial}{\partial \theta*0} J(\theta_0, \theta_1) = \frac{1}{m} \sum*{i=1}^{m} (h\_\theta(x_i) - y_i) $$
2.  **Derivative for $\theta_1$ ($j=1$):**
    $$ \frac{\partial}{\partial \theta*1} J(\theta_0, \theta_1) = \frac{1}{m} \sum*{i=1}^{m} (h\_\theta(x_i) - y_i) \cdot x_i $$

**The Golden Rule of Updates:**
You **must** update $\theta_0$ and $\theta_1$ _simultaneously_.

- Calculate `temp0` using the old $\theta_0$.
- Calculate `temp1` using the old $\theta_1$.
- Set $\theta_0 = temp0$ and $\theta_1 = temp1$.
  If you update $\theta_0$ first, and then use the _new_ $\theta_0$ to calculate the gradient for $\theta_1$, your gradient descent will be mathematically incorrect and may fail to converge.

### Feature Scaling for Optimization

If variables are on completely different scales (e.g., Number of Bedrooms [1-5] vs. Square Footage [500-5000]), gradient descent will create an elongated, elliptical bowl, causing the algorithm to bounce inefficiently.

- **Min-Max Scaling:** $x_{ij} = \frac{x_{ij} - Min_j}{Max_j - Min_j}$ (Forces values between $[0, 1]$).
- **Standardization (Z-score):** $x_{ij} = \frac{x_{ij} - \mu_j}{\sigma_j}$ (Centers around 0. Approximates values between $[-3\sigma, +3\sigma]$).
