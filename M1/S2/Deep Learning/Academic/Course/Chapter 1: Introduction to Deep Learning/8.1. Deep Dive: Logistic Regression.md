# 8.1. Deep Dive: Logistic Regression

Logistic Regression is a **Classification** technique.

- Binary Classification: $y \in \{0, 1\}$ (e.g., Spam/Not Spam, Malignant/Benign tumor).
- Multi-class: $y \in \{0, 1, 2, 3\}$.

### The Sigmoid Function

We need our hypothesis to output a value between 0 and 1 (a probability). We do this by passing our linear equation ($\theta^T x$) through the Sigmoid (logistic) function:
$$ g(z) = \frac{1}{1 + e^{-z}} $$
$$ h(x) = g(\theta^T x) = \frac{1}{1 + e^{-\theta^T x}} $$

- If $z \to +\infty$, $g(z) \to 1$.
- If $z \to -\infty$, $g(z) \to 0$.

### The Decision Boundary

The model predicts Class 1 if the probability is $\ge 0.5$.
Looking at the sigmoid curve, $g(z) \ge 0.5$ whenever $z \ge 0$.
Therefore, the decision boundary is the line where **$\theta^T x = 0$**.
_Note: The decision boundary can be linear, or if we use polynomial features (like $x_1^2 + x_2^2$), the boundary can be a complex curve (like a circle)._

### Why MSE Fails for Logistic Regression

If we substitute $h(x) = \frac{1}{1 + e^{-\theta^T x}}$ into the standard MSE cost function:
$$ J(\theta) = \frac{1}{2m} \sum (\frac{1}{1 + e^{-\theta^T x}} - y)^2 $$
This creates a **Non-Convex** function. It will have numerous hills and valleys (local minima). Gradient Descent cannot be trusted on this landscape.

### The Log Loss (Binary Cross-Entropy)

To restore a perfect, convex "bowl" shape, we redefine the cost function using logarithms:

- Cost if $y=1$: $-\log(h(x))$
- Cost if $y=0$: $-\log(1 - h(x))$

We combine these into a single, elegant vectorized equation:
$$ J(\theta) = -\frac{1}{m} \sum*{i=1}^{m} [y^{(i)} \log(h*\theta(x^{(i)})) + (1 - y^{(i)}) \log(1 - h\_\theta(x^{(i)}))] $$

**The Beautiful Mathematical Coincidence:**
When you take the derivative of this Log Loss function, the complex derivative of the Sigmoid mathematically cancels out elements of the log derivative. The resulting gradient calculation is:
$$ \frac{\partial J(\theta)}{\partial \theta*j} = \frac{1}{m} \sum*{i=1}^{m} (h*\theta(x^{(i)}) - y^{(i)}) x_j^{(i)} $$
**This is the EXACT SAME formula as Linear Regression.** However, remember that $h*\theta(x)$ is different (Sigmoid vs. Linear).
