
# 2. The Canonical Tableau Structure

This note establishes the **Standard Tableau**. Many tutorials skip columns to save space, which causes confusion. We will use the **Full Explicit Form** so you see where every number comes from.

## **2.1. The Setup**
Assume a Maximization problem.
**Objective Function:** $\text{Max } Z = c_1x_1 + c_2x_2 + \dots + c_nx_n$
**Subject to:** $Ax \le b$

We convert to **Standard Form** by adding Slack variables ($s_1, s_2, \dots$) with coefficients of $0$ in the Objective Function.

## **2.2. The Master Tableau Layout**

| $C_B$ | Basis | $x_1$ | $x_2$ | $s_1$ | $s_2$ | $b$ (RHS) | Ratio ($\theta$) |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| **(1)** | **(2)** | **(3)** | **(3)** | **(3)** | **(3)** | **(4)** | **(5)** |
| $c_{s1}$ | $s_1$ | $a_{11}$ | $a_{12}$ | 1 | 0 | $b_1$ | $b_1 / a_{1k}$ |
| $c_{s2}$ | $s_2$ | $a_{21}$ | $a_{22}$ | 0 | 1 | $b_2$ | $b_2 / a_{2k}$ |
| **$Z_j$** | **(6)** | $\sum (C_B \cdot x_1)$ | $\sum (C_B \cdot x_2)$ | $\dots$ | $\dots$ | **Current Z** | |
| **$C_j - Z_j$** | **(7)** | $c_1 - z_1$ | $c_2 - z_2$ | $0 - z_{s1}$| $0 - z_{s2}$| | |

### **Explanation of Columns**
*   **Row $C_j$ (Top Header):** Contains the original coefficients from the Objective Function ($25, 15, 0, 0$ from File 2). **These never change.**
*   **Col (1) $C_B$:** The Objective Coefficient of the variable *currently in the Basis*.
*   **Col (2) Basis:** The names of the variables currently selected (Basic Variables). Initially, these are always the Slack Variables ($s_1, s_2$).
*   **Col (3) Matrix A:** The coefficients from the constraints.
*   **Col (4) $b$ / RHS:** The Quantity/Solution column. The value of the Basic variables.
*   **Col (5) Ratio:** Used only to decide which row leaves. Calculated as $b / \text{Pivot Column Element}$.
*   **Row (6) $Z_j$:** The "Cost Incurred" row. Calculated by multiplying Column (1) by Column (3) and summing.
    *   *Example:* $Z_{x1} = (c_{s1} \times a_{11}) + (c_{s2} \times a_{21})$.
*   **Row (7) $C_j - Z_j$:** The "Net Evaluation" row.
    *   *Calculation:* Top Header Value - Row (6) Value.
