# Course Syllabus: Advanced Databases

## Objectives
This course aims to master the design, management, and exploitation of modern database systems. It covers the gap between basic SQL and enterprise-grade database administration.

## Prerequisites
1.  **Relational Model:** Understanding Tables, Keys, and Normalization.
2.  **SQL:** Proficiency in `SELECT`, `JOIN`, `GROUP BY`, `INSERT`, `UPDATE`.
3.  **Relational Algebra:** Selection, Projection, Cartesian Product.

## Detailed Program

### Chapter 1: Introduction & Fundamentals
*   Recap of Relational Model & SGBD.
*   Types of DBs: NoSQL, Object-Oriented, Multimedia, Time-series.
*   Architecture: Client-Server vs. Distributed.
*   Transaction concepts (ACID).

### Chapter 2: Advanced SQL Programming
*   Joins, Sub-queries, Aggregates.
*   **Advanced Objects:** Stored Procedures, Views, Triggers, Functions.
*   Error Handling in SQL.
*   NoSQL Querying (JSONiq, Cypher).

### Chapter 3: Object-Relational Model
*   Integrating OOP concepts into SQL.
*   Complex types, Inheritance in databases.
*   SQL3 standards.

### Chapter 4: NoSQL Databases
*   Key-Value, Document, Column-Family, Graph stores.
*   Case studies: MongoDB, Cassandra, Neo4j.

### Chapter 5: Distributed Databases
*   Architecture types (Shared nothing, shared disk).
*   Replication & Partitioning (Sharding).
*   CAP Theorem (ACID vs BASE).

### Chapter 6: Cloud Databases
*   IaaS vs PaaS vs SaaS database models.
*   Cloud security considerations.

### Chapter 7: Performance & Security
*   Query Optimization.
*   Indexing strategies (B-Tree, Hash).
*   Access Control and Auditing.
