# Test Question 10 (Multiple Choice Math Options)

## Question

What is the solution to the quadratic equation $ax^2 + bx + c = 0$?

## Options

- $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$
- $x = \frac{-b \pm \sqrt{b^2 + 4ac}}{2a}$
- $x = \frac{b \pm \sqrt{b^2 - 4ac}}{2a}$
- $x = -b \pm \sqrt{b^2 - 4ac}$

## Answer

$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$
