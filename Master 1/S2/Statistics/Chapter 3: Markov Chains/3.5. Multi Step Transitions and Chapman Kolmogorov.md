# 3.5. Multi-Step Transitions and Chapman-Kolmogorov

### 1. Multi-Step Transition Probabilities
The probability that a Markov chain transitions from state $a_i$ to state $a_j$ in exactly $n$ steps is written as $P_{ij}^{(n)}$:
$$P_{ij}^{(n)} = P(X_{k+n} = a_j \mid X_k = a_i)$$

### 2. Theorem: Matrix Powers
The $n$-step transition matrix $P^{(n)}$ is simply the $n$-th power of the one-step transition matrix $P$:
$$P^{(n)} = P^n$$

* **Intuition:** To go from state $i$ to state $j$ in $n$ steps, the system must pass through some intermediate state $r$ at step $n-1$. Summing over all possible intermediate states yields:
  $$P_{ij}^{(n)} = \sum_{r \in S} P_{ir}^{(n-1)} P_{rj}$$
  This is the exact formula for matrix multiplication, meaning $P^{(n)} = P^{(n-1)} \cdot P = P^n$.

### 3. State Distribution Evolution
Let $p^{(0)} = (P(X_0 = a_1), \dots, P(X_0 = a_m))$ be the probability distribution of the initial state of the system. 

Using the transition matrix, we can calculate the probability distribution of the states after $n$ steps:
$$p^{(n)} = p^{(0)} P^n$$
