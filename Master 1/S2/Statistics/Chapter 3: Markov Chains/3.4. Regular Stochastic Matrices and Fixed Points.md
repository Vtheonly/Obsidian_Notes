# 3.4. Regular Stochastic Matrices and Fixed Points

### 1. Probability Vectors
A row vector $u = (u_1, u_2, \dots, u_m)$ is a **probability vector** if it satisfies two conditions:
1. All elements are non-negative: $u_i \ge 0$ for all $i$.
2. The elements sum to 1: $\sum_{i=1}^{m} u_i = 1$.

#### Proposition 3.1: Normalization
Any vector $v = (v_1, v_2, \dots, v_m)$ with strictly positive elements can be converted into a probability vector $u$ by dividing each element by the sum of all elements:
$$u_i = \frac{v_i}{\sum_{k=1}^{m} v_k}$$

---

### 2. Regular Stochastic Matrices
A stochastic matrix $P$ is **regular** if there exists some positive integer $n_0 \ge 1$ such that all elements of the power matrix $P^{n_0}$ are strictly positive:
$$(P^{n_0})_{ij} > 0 \quad \text{for all } i, j$$
This means that after $n_0$ steps, it is possible to transition from any state to any other state.

#### Proposition 3.2: Diagonal Elements of 1
If a stochastic matrix of size $m \times m$ (where $m \ge 2$) contains a $1$ on its main diagonal (e.g., $P_{ii} = 1$), it is **not regular**.

* **Proof:** If $P_{ii} = 1$, then once the system enters state $a_i$, it can never leave:
  $$P_{ii} = 1 \implies P_{ij} = 0 \quad \text{for all } j \neq i$$
  Calculating any power of this matrix, $P^n$ will always have $(P^n)_{ii} = 1$ and $(P^n)_{ij} = 0$ for all $j \neq i$. 
  Because these zero elements persist for all powers $n$, there is no $n_0$ where all elements of $P^{n_0}$ are strictly positive. Thus, the matrix is not regular.

---

### 3. Stationary Distributions (Fixed Points)

#### Theorem 3.1: The Stationary Distribution Theorem
If $P$ is a regular stochastic matrix, then:
1. There exists a unique probability vector $t = (t_1, t_2, \dots, t_m)$ (called the **stationary distribution** or **fixed point**) such that:
   $$tP = t$$
   where $t_i > 0$ for all $i$.
2. The sequence of powers $P^n$ converges to a limiting matrix $T$ where every row is equal to the stationary vector $t$:
   $$\lim_{n \to +\infty} P^n = T = \begin{pmatrix}
   t_1 & t_2 & \dots & t_m \\
   t_1 & t_2 & \dots & t_m \\
   \vdots & \vdots & \ddots & \vdots \\
   t_1 & t_2 & \dots & t_m
   \end{pmatrix}$$
3. For any initial probability vector $p^{(0)}$, the state distribution converges to $t$ in the long run:
   $$\lim_{n \to +\infty} p^{(0)} P^n = t$$

### 4. Step-by-Step Stationary Vector Calculations

#### Example A: A $2 \times 2$ Regular Matrix
Let's find the stationary distribution for the transition matrix:
$$P = \begin{pmatrix} 0 & 1 \\ 1/2 & 1/2 \end{pmatrix}$$

We define our stationary vector as $t = (x, 1-x)$ to ensure its components sum to 1. We then solve the equation $tP = t$:
$$(x, \quad 1-x) \begin{pmatrix} 0 & 1 \\ 1/2 & 1/2 \end{pmatrix} = (x, \quad 1-x)$$

Multiplying the vector by the matrix yields the system of equations:
1. For the first component:
   $$x \cdot 0 + (1-x) \cdot \frac{1}{2} = x \implies \frac{1-x}{2} = x$$
2. Multiplying both sides by 2:
   $$1 - x = 2x \implies 3x = 1 \implies x = \frac{1}{3}$$
3. The second component is then:
   $$1 - x = 1 - \frac{1}{3} = \frac{2}{3}$$

This gives us the unique stationary distribution:
$$t = \left(\frac{1}{3}, \frac{2}{3}\right) \approx (0.3333, \quad 0.6667)$$

---

#### Example B: A $3 \times 3$ Regular Matrix
Now let's find the stationary distribution for this transition matrix:
$$P = \begin{pmatrix} 0 & 1 & 0 \\ 0 & 0 & 1 \\ 1/2 & 1/2 & 0 \end{pmatrix}$$

We define the stationary vector as $t = (x, y, 1-x-y)$ and solve $tP = t$:
$$(x, \quad y, \quad 1-x-y) \begin{pmatrix} 0 & 1 & 0 \\ 0 & 0 & 1 \\ 1/2 & 1/2 & 0 \end{pmatrix} = (x, \quad y, \quad 1-x-y)$$

Multiplying the vector by the matrix gives us:
$$\left( \frac{1}{2}(1-x-y), \quad x + \frac{1}{2}(1-x-y), \quad y \right) = (x, \quad y, \quad 1-x-y)$$

This yields three equations:
1. $\frac{1}{2}(1-x-y) = x \implies 1-x-y = 2x \implies 3x + y = 1$
2. $x + \frac{1}{2}(1-x-y) = y$
3. $y = 1-x-y \implies x + 2y = 1$

We can solve this system using the first and third equations:
* From equation 1, we isolate $y$:
  $$y = 1 - 3x$$
* Substituting this into equation 3:
  $$x + 2(1 - 3x) = 1 \implies x + 2 - 6x = 1 \implies -5x = -1 \implies x = \frac{1}{5}$$
* Now we find $y$:
  $$y = 1 - 3\left(\frac{1}{5}\right) = \frac{2}{5}$$
* Finally, we calculate the third component:
  $$1 - x - y = 1 - \frac{1}{5} - \frac{2}{5} = \frac{2}{5}$$

This gives us the unique stationary distribution:
$$t = \left(\frac{1}{5}, \frac{2}{5}, \frac{2}{5}\right) = (0.2, \quad 0.4, \quad 0.4)$$

---

#### Alternative Method: Normalization of Free Variables
We can also solve this by using three variables $t = (x, y, z)$ and applying the normalization condition $x + y + z = 1$ at the end.

Using the equation $tP = t$:
$$(x, \quad y, \quad z) \begin{pmatrix} 0 & 1 & 0 \\ 0 & 0 & 1 \\ 1/2 & 1/2 & 0 \end{pmatrix} = (x, \quad y, \quad z)$$

This gives us the system:
1. $\frac{1}{2}z = x \implies z = 2x$
2. $x + \frac{1}{2}z = y \implies x + x = y \implies y = 2x$
3. $y = z \implies 2x = 2x$ (which is consistent)

We can express all variables in terms of $x$:
$$t = (x, \quad 2x, \quad 2x)$$
To find the probability vector, we normalize by dividing each component by the sum of all components ($x + 2x + 2x = 5x$):
$$t = \left( \frac{x}{5x}, \quad \frac{2x}{5x}, \quad \frac{2x}{5x} \right) = \left(\frac{1}{5}, \frac{2}{5}, \frac{2}{5}\right)$$
This yields the same result.
