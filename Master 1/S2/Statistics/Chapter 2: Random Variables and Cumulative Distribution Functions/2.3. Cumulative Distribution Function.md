# 2.3. Cumulative Distribution Function

### 1. Formal Definition
The **Cumulative Distribution Function (CDF)** (denoted by $F_X(x)$) of a real random variable $X$ is the probability that $X$ takes a value less than or equal to $x$:
$$\forall x \in \mathbb{R}, \quad F_X(x) = P(X \le x) = P(\{\omega \in \Omega \mid X(\omega) \le x\})$$

For a **discrete** random variable, we calculate the CDF by summing the probabilities of all values less than or equal to $x$:
$$F_X(x) = \sum_{x_i \le x} P(X = x_i)$$

### 2. Core Mathematical Properties
Any valid CDF $F(x)$ must satisfy the following four mathematical properties:

1. **Monotonically Non-Decreasing:** If $x_1 < x_2$, then $F(x_1) \le F(x_2)$.
2. **Right-Continuity:** The function is continuous when approached from the right:
   $$\lim_{t \to x^+} F(t) = F(x)$$
3. **Lower Limit at $-\infty$:**
   $$\lim_{x \to -\infty} F(x) = 0$$
4. **Upper Limit at $+\infty$:**
   $$\lim_{x \to +\infty} F(x) = 1$$

### 3. Step-by-Step Step Function Example
Let's construct the step function CDF from the course material.

* **Given Distribution:**
  $$P(X=0) = 0.125, \quad P(X=1) = 0.375, \quad P(X=2) = 0.375, \quad P(X=3) = 0.125$$

#### Constructing $F_X(x)$ for All Real Numbers
* **Case 1: $x < 0$**
  There are no values of $X$ in this range.
  $$F_X(x) = 0$$
* **Case 2: $0 \le x < 1$**
  Only the value $X=0$ lies in this interval.
  $$F_X(x) = P(X=0) = 0.125$$
* **Case 3: $1 \le x < 2$**
  The values $X=0$ and $X=1$ lie in this interval.
  $$F_X(x) = P(X=0) + P(X=1) = 0.125 + 0.375 = 0.500$$
* **Case 4: $2 \le x < 3$**
  The values $X=0, 1, 2$ lie in this interval.
  $$F_X(x) = P(X=0) + P(X=1) + P(X=2) = 0.500 + 0.375 = 0.875$$
* **Case 5: $x \ge 3$**
  All possible values of $X$ lie in this interval.
  $$F_X(x) = P(X=0) + P(X=1) + P(X=2) + P(X=3) = 0.875 + 0.125 = 1.000$$

Combining these cases, we get the complete CDF:
$$F_X(x) = \begin{cases} 
0 & \text{if } x < 0 \\ 
0.125 & \text{if } 0 \le x < 1 \\ 
0.5 & \text{if } 1 \le x < 2 \\ 
0.875 & \text{if } 2 \le x < 3 \\ 
1 & \text{if } x \ge 3 
\end{cases}$$
