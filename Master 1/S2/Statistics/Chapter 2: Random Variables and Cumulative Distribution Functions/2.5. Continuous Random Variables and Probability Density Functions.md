# 2.5. Continuous Random Variables and Probability Density Functions

### 1. Absolute Continuity
A random variable $X$ is **absolutely continuous** if there exists a non-negative, integrable function $f_X: \mathbb{R} \to [0, +\infty)$ such that its Cumulative Distribution Function can be expressed as:
$$F_X(x) = P(X \le x) = \int_{-\infty}^{x} f_X(t) \, dt$$
The function $f_X(t)$ is called the **Probability Density Function (PDF)** of $X$.

To be a valid PDF, $f(x)$ must satisfy two conditions:
1. **Non-negativity:** $f(x) \ge 0$ for all $x \in \mathbb{R}$.
2. **Total Area of 1:** The integral over the entire real line must equal 1:
   $$\int_{-\infty}^{+\infty} f(t) \, dt = 1$$

### 2. Core Properties of Continuous Random Variables

#### Property 1: Probability at any single point is zero
$$\forall a \in \mathbb{R}, \quad P(X = a) = 0$$
* **Mathematical Proof:**
  Using the properties of integrals:
  $$P(X = a) = P(a \le X \le a) = \int_{a}^{a} f(t) \, dt = 0$$
* **Conceptual Intuition:** In a continuous space, there are infinitely many possible outcomes. The probability of hitting any single exact value (such as measuring a temperature of exactly $23.1234567\dots^\circ\text{C}$ with infinite precision) is infinitely small, and thus zero. Instead, we calculate probabilities over intervals of values.

#### Property 2: Interval Probabilities
$$P(a \le X \le b) = \int_{a}^{b} f(t) \, dt = F(b) - F(a)$$
Because the probability at any single point is zero, the inclusion or exclusion of the endpoints does not change the probability of the interval:
$$P(a \le X \le b) = P(a < X \le b) = P(a \le X < b) = P(a < X < b)$$

#### Property 3: Continuous CDF
If $X$ is a continuous random variable, its CDF $F_X(x)$ is a continuous, strictly increasing function.

#### Property 4: Derivative Relationship
According to the Fundamental Theorem of Calculus, if the PDF $f(x)$ is continuous at a point $x$, then the PDF is the derivative of the CDF:
$$\frac{d}{dx} F_X(x) = f_X(x)$$
