# 2.4. Classical Discrete Distributions and Poisson Recurrence

### 1. Poisson Distribution $\mathcal{P}(\lambda)$
The Poisson distribution models the number of events occurring within a fixed interval of time or space, assuming these events occur with a known constant average rate and independently of the time since the last event.

* **Support:** $X(\Omega) = \mathbb{N} = \{0, 1, 2, 3, \dots\}$
* **Parameter:** $\lambda > 0$ (the average rate of occurrence)
* **Probability Mass Function (PMF):**
  $$P(X = k) = \frac{e^{-\lambda} \lambda^k}{k!} \quad \text{for } k \in \{0, 1, 2, \dots\}$$

#### Crucial Correction: Poisson Recurrence Relation
The course slides list a recurrence relation under the geometric distribution section. Let's clarify: this relation actually belongs to the **Poisson distribution**. 

The relation allows us to calculate Poisson probabilities recursively:
$$P(X = k) = \frac{\lambda}{k} P(X = k-1) \quad \text{for } k \ge 1$$

* **Algebraic Proof:**
  We write out the expression for $P(X = k-1)$:
  $$P(X = k-1) = \frac{e^{-\lambda} \lambda^{k-1}}{(k-1)!}$$
  Now, we multiply this expression by $\frac{\lambda}{k}$:
  $$\frac{\lambda}{k} P(X = k-1) = \frac{\lambda}{k} \cdot \frac{e^{-\lambda} \lambda^{k-1}}{(k-1)!} = \frac{e^{-\lambda} \lambda^{k-1+1}}{k \cdot (k-1)!}$$
  Since $k \cdot (k-1)! = k!$, this simplifies to:
  $$\frac{\lambda}{k} P(X = k-1) = \frac{e^{-\lambda} \lambda^k}{k!} = P(X = k)$$
  This completes the proof.

### 2. Geometric Distribution $\mathcal{G}(p)$
The geometric distribution models the number of independent Bernoulli trials needed to get the first success, where each trial has a success probability $p$.

* **Support:** $X(\Omega) = \{1, 2, 3, \dots\}$
* **Parameter:** $p \in (0, 1]$
* **Probability Mass Function (PMF):**
  $$P(X = k) = p(1-p)^{k-1} \quad \text{for } k \in \{1, 2, 3, \dots\}$$

* **Alternative Form (Support starting at 0):** If we define $Y$ as the number of failures *before* the first success (where $Y = X - 1$, so $Y \in \{0, 1, 2, \dots\}$), the PMF is:
  $$P(Y = k) = p(1-p)^k \quad \text{for } k \in \{0, 1, 2, \dots\}$$
