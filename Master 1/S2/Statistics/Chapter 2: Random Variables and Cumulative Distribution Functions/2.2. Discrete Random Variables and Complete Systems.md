# 2.2. Discrete Random Variables and Complete Systems

### 1. Probability Distribution of a Discrete Random Variable
For a discrete random variable $X$, its distribution is completely defined by its **Probability Mass Function (PMF)**:
$$p_i = P(X = x_i) = P(\{\omega \in \Omega \mid X(\omega) = x_i\})$$

Since the set of all possible outcomes $X(\Omega) = \{x_1, x_2, \dots\}$ is countable, the preimages of these values partition the sample space:
$$X^{-1}(\{x_i\}) \cap X^{-1}(\{x_j\}) = \emptyset \quad \text{for } i \neq j, \quad \text{and} \quad \bigcup_{i} X^{-1}(\{x_i\}) = \Omega$$
Therefore, the collection of events $(X = x_i)_i$ forms a **complete system of events**, which guarantees that their probabilities sum to 1:
$$\sum_{i} P(X = x_i) = 1$$

### 2. Detailed Modulo 3 Die Roll Example
Let's analyze the modulo 3 die roll example from the course material.

* **The Experiment:** Roll a standard fair six-sided die.
  $$\Omega = \{1, 2, 3, 4, 5, 6\}$$
* **The Probability Measure:** Since the die is fair, each outcome has an equal probability:
  $$\forall \omega \in \Omega, \quad P(\{\omega\}) = \frac{1}{6}$$
* **The Specific $\sigma$-Algebra ($\mathcal{A}$):**
  $$\mathcal{A} = \{\emptyset, \{1,4\}, \{2,5\}, \{3,6\}, \{1,2,4,5\}, \{1,3,4,6\}, \{2,3,5,6\}, \Omega\}$$
* **The Random Variable ($X$):** Define $X$ as the remainder when the outcome $\omega$ is divided by 3:
  $$X(\omega) = \omega \pmod 3$$

#### Step-by-Step Evaluation of $X(\omega)$
$$\begin{aligned}
X(1) &= 1 \pmod 3 = 1 \\
X(2) &= 2 \pmod 3 = 2 \\
X(3) &= 3 \pmod 3 = 0 \\
X(4) &= 4 \pmod 3 = 1 \\
X(5) &= 5 \pmod 3 = 2 \\
X(6) &= 6 \pmod 3 = 0
\end{aligned}$$
This gives us the range of $X$:
$$X(\Omega) = {0, 1, 2}$$

#### Verifying the Measurability of $X$
To confirm that $X$ is a mathematically valid random variable under the given $\sigma$-algebra $\mathcal{A}$, we must verify that the preimage of every value in its range belongs to $\mathcal{A}$:
* For $0$: $X^{-1}(\{0\}) = \{\omega \in \Omega \mid X(\omega) = 0\} = \{3, 6\}$. Looking at our $\sigma$-algebra, we see that $\{3,6\} \in \mathcal{A}$.
* For $1$: $X^{-1}(\{1\}) = \{\omega \in \Omega \mid X(\omega) = 1\} = \{1, 4\}$. This also belongs to the $\sigma$-algebra: $\{1,4\} \in \mathcal{A}$.
* For $2$: $X^{-1}(\{2\}) = \{\omega \in \Omega \mid X(\omega) = 2\} = \{2, 5\}$. This also belongs to the $\sigma$-algebra: $\{2,5\} \in \mathcal{A}$.

Since the preimage of every single point in the range belongs to $\mathcal{A}$, $X$ is indeed a valid discrete real random variable under this $\sigma$-algebra.

#### Calculating the Probabilities
Using the equal probability distribution of the outcomes, we calculate the probability for each value of $X$:
$$P(X = 0) = P(\{3, 6\}) = P(\{3\}) + P(\{6\}) = \frac{1}{6} + \frac{1}{6} = \frac{1}{3}$$
$$P(X = 1) = P(\{1, 4\}) = P(\{1\}) + P(\{4\}) = \frac{1}{6} + \frac{1}{6} = \frac{1}{3}$$
$$P(X = 2) = P(\{2, 5\}) = P(\{2\}) + P(\{5\}) = \frac{1}{6} + \frac{1}{6} = \frac{1}{3}$$

The sum of these probabilities is indeed 1:
$$\sum_{k=0}^{2} P(X = k) = \frac{1}{3} + \frac{1}{3} + \frac{1}{3} = 1$$

This confirms that the events $(X=0)$, $(X=1)$, and $(X=2)$ form a complete system of events.
