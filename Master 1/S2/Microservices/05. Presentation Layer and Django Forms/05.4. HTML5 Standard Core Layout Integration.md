
# 5.4. HTML5 Standard Core Layout Integration

## 1. Designing a Professional Semantic Base Layout
To ensure your web applications are accessible, search-engine-friendly, and easy to maintain, you should use standard HTML5 semantic elements (like `<header>`, `<nav>`, `<main>`, `<section>`, and `<footer>`).

Below is a production-ready, accessible HTML5 base layout configuration utilizing Django Template Language placeholders.

### Layout File Path: `templates/base.html`
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <!-- Core Character Set -->
    <meta charset="UTF-8">
    <!-- Responsive Viewport Scale Configurations -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    
    <!-- Meta Search Descriptions -->
    <meta name="description" content="{% block meta_description %}Secure Clinical Records Management Portal{% endblock %}">
    
    <!-- Page Title Dynamic Override -->
    <title>{% block title %}Enterprise Clinical Space{% endblock %}</title>
    
    <!-- Shared Global Static Stylesheets -->
    {% load static %}
    <link rel="stylesheet" href="{% static 'css/normalize.css' %}">
    <link rel="stylesheet" href="{% static 'css/core-theme.css' %}">
    
    <!-- Child Page-Specific CSS Overrides -->
    {% block extra_css %}{% endblock %}
</head>
<body>

    <!-- Accessible Skip to Content Link -->
    <a href="#main-content" class="skip-link">Skip to main content</a>

    <!-- Core Header Identity Block -->
    <header class="site-header">
        <div class="brand">
            <img src="{% static 'images/sys-emblem.svg' %}" alt="Portal Brand Icon" class="logo">
            <span class="system-title">Clinical Operations Node</span>
        </div>
        
        <!-- Primary Navigation Links -->
        <nav class="primary-navigation" aria-label="Main system navigation">
            <ul>
                <li><a href="{% url 'clinical:dashboard' %}" class="nav-item">System Monitor</a></li>
                <li><a href="{% url 'clinical:patients-list' %}" class="nav-item">Registries</a></li>
                <li><a href="/accounts/logout/" class="nav-item">Log Out</a></li>
            </ul>
        </nav>
    </header>

    <!-- Unified Main Layout Space -->
    <main id="main-content" class="container" tabindex="-1">
        
        <!-- Flash Alert Messages Container -->
        {% if messages %}
            <aside class="alerts-wrapper" aria-label="System notification updates">
                {% for message in messages %}
                    <div class="alert alert-{{ message.tags }}" role="status">
                        {{ message }}
                    </div>
                {% endfor %}
            </aside>
        {% endif %}

        <!-- Dynamic Content Body Block -->
        <article class="content-wrapper">
            {% block content %}
                <h1>System Warning</h1>
                <p>Welcome to the empty core layout node.</p>
            {% endblock %}
        </article>
        
    </main>

    <!-- Global Layout Footer -->
    <footer class="system-footer">
        <p class="copyright-text">&copy; 2026 Node Corp. Confidential Medical Datastores.</p>
    </footer>

    <!-- Shared Core JavaScript Engines -->
    <script src="{% static 'js/system-runtime.js' %}"></script>
    
    <!-- Child Page-Specific Script Injection -->
    {% block extra_js %}{% endblock %}

</body>
</html>
```
