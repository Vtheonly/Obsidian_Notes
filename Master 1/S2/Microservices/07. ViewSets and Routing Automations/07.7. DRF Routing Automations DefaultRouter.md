
# 7.7. DRF Routing Automations DefaultRouter

## 1. SimpleRouter vs. DefaultRouter
Django REST Framework provides two primary router classes: `SimpleRouter` and `DefaultRouter`.

While both routers generate standard URL patterns for your ViewSets, **`DefaultRouter`** includes two additional features:
1. **The API Root View**: Generates a default, interactive landing page at the root URL path (`/`) that displays a list of all your registered API endpoints.
2. **Format Suffix Patterns**: Adds support for format suffixes on URLs (e.g., accessing `/api/patients.json` to force a JSON response, or `/api/patients.api` to view the browsable web interface).

## 2. Python Implementation Example
```python
# File: clinical/urls.py
from rest_framework.routers import DefaultRouter
from .views import PatientModelViewSet, AdvancedPatientViewSet

# 1. Instantiate the DefaultRouter
router = DefaultRouter()

# 2. Register your application's ViewSets
router.register(r'patients', PatientModelViewSet, basename='patient')
router.register(r'advanced-patients', AdvancedPatientViewSet, basename='adv-patient')

# 3. Include the router's generated URL patterns
urlpatterns = router.urls
```

## 3. Registering App Routers with the Project's Main URLs
In a standard Django project, you should organize your routing by defining local routers inside each app's `urls.py` file, and then importing and combining them in your project's main `urls.py` configuration file.

### Project-Level Main URL Configuration: `project_core/urls.py`
```python
from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    # Django Admin Panel
    path('admin/', admin.site.urls),
    
    # Include your app's local API routes under the '/api/' namespace prefix
    path('api/', include('clinical.urls')),
]
```

Under this configuration, your API endpoints are accessible at these paths:
```http
GET /api/                    --> Interactive API Root View
GET /api/patients/           --> Patient Registry List
GET /api/patients.json       --> Explicit JSON formatted List
```
