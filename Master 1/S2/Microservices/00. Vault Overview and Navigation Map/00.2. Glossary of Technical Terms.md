# 00.2. Glossary of Technical Terms

> [!abstract] Overview
> A centralized dictionary of recurring architectural, networking, and programming terms used throughout this vault.

### Architecture Paradigms
- **Monolith**: A software application where the user interface, business rules, and data interface are combined into a single, unified, and tightly coupled executable program.
- **Microservices**: An architectural style that structures an application as a collection of loosely coupled, independently deployable services organized around business capabilities.
- **MVT (Model-View-Template)**: A software design pattern used by Django. It separates the data access logic (Model), the business/controller logic (View), and the presentation markup (Template).

### Security and Identity
- **JWT (JSON Web Token)**: A stateless, cryptographically signed token used to transmit identity and claims securely between a client and a server.
- **Authentication**: The process of verifying *who* a user is (e.g., verifying an email and password).
- **Authorization**: The process of verifying *what* an authenticated user is allowed to do (e.g., checking if they have `can_publish` permissions).

### Distributed Systems and Infrastructure
- **API Gateway**: A server that acts as an API front-end, receiving API requests, enforcing throttling and security policies, passing requests to the back-end service, and then passing the response back to the requester.
- **Service Discovery**: The automatic detection of devices and services offered by these devices on a computer network. 
- **Consul**: A service networking solution developed by HashiCorp used for service discovery, health checking, and service mesh routing.
- **Raft**: A consensus algorithm used by Consul servers to elect a leader and maintain a synchronized, consistent state across the cluster.
- **Gossip Protocol (Serf)**: A peer-to-peer communication protocol used by Consul agents to broadcast messages (like node failures or membership changes) rapidly across a network without a central bottleneck.
