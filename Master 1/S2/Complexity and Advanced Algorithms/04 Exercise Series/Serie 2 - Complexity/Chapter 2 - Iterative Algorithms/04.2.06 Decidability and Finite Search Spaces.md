# 4. Decidability and Finite Search Spaces

Based on the handwritten notes in Ex 7.

## The Problem
The classical problem asks: "Determine if there exists a sequence of indices of length $m \ge 1$ such that the product of the matrix elements is zero."

### Infinite Space = Undecidable
If the length of the sequence $m$ is unbounded ($m$ can be anything to infinity), the problem space is infinite. We cannot guarantee that an algorithm will finish searching. Therefore, the general version of this problem is **undecidable** (indécidable).

### Finite Space = Decidable
The exercise modifies the constraint: $1 \le m \le 1000$. 
This changes everything. By restricting $m$ to a maximum of 1000, we restrict the search to a **finite set of combinations**.

Let's calculate the total number of combinations to check:
* For a sequence of length $m=1$: There are $K$ possible placements (choices).
* For a sequence of length $m=2$: For the first placement, we have $K$ choices. For the second, $K$ choices. Total = $K \times K = K^2$ possibilities.
* ...
* For a sequence of length $m=1000$: Total = $K^{1000}$ possibilities.

The total number of products to verify is the sum of all possibilities for every allowed length:
$$ N = K^1 + K^2 + K^3 + \dots + K^{1000} = \sum_{i=1}^{1000} K^i $$

Because this set of combinations $N$ is finite (it's a very large number, but it is not infinity), a computer can theoretically check all of them and halt.
**Conclusion:** Because the search space is finite, the problem is **decidable**.
