# 2. Average Case Calculation using Probability

> [!warning] Essential Context
> Calculating average case complexity ($T_{moy}$) is heavily dependent on probability. You cannot simply average the best and worst case. You must calculate the mathematical expectation: the sum of the cost of every possible outcome, multiplied by the probability of that outcome occurring.

## The Problem Setup
We are still analyzing the matrix search algorithm (from Note 1). We want to find the average number of operations to find $x$.

### 1. Defining Uniformity (Probability)
The slide states the assumption of **Uniformity**. 
This means that the item $x$ has an absolutely equal chance of being in *any* cell of the matrix. 
- Total cells in an $n \times n$ matrix = $n^2$.
- Probability of $x$ being in a specific cell $k$ = $\frac{1}{n^2}$.

### 2. Defining Cost $C(k)$
Let $k$ represent the position number of a cell as we read the matrix from left to right, top to bottom.
- 1st position $(1,1) \rightarrow k=1 \rightarrow \text{Requires 1 comparison: } C(1) = 1$
- 2nd position $(1,2) \rightarrow k=2 \rightarrow \text{Requires 2 comparisons: } C(2) = 2$
- ...
- $k^{th}$ position $\rightarrow \text{Requires k comparisons: } C(k) = k$

### 3. The Expected Value Formula
The average time ($T_{moy}$) is defined mathematically as:
$$T_{moy} = \sum_{k=1}^{n^2} (\text{Probability of position } k) \times (\text{Cost of position } k)$$

Substitute our established values:
$$T_{moy} = \sum_{k=1}^{n^2} \left( \frac{1}{n^2} \right) \times k$$

Since $\frac{1}{n^2}$ is a constant relative to $k$, we can pull it outside the summation:
$$T_{moy} = \frac{1}{n^2} \sum_{k=1}^{n^2} k$$

> [!info] Mathematical Reminder: Gauss's Summation Trick
> You must memorize the arithmetic progression sum formula. The sum of integers from $1$ to $N$ is:
> $\sum_{k=1}^{N} k = \frac{N(N + 1)}{2}$
> In our case, our upper limit $N$ is $n^2$. So replacing $N$ with $n^2$ yields:
> $\sum_{k=1}^{n^2} k = \frac{n^2(n^2 + 1)}{2}$

### 4. Final Calculation
Substitute the arithmetic progression formula back into our main equation:

$$T_{moy} = \frac{1}{n^2} \times \left[ \frac{n^2(n^2 + 1)}{2} \right]$$

The $n^2$ terms cancel each other out:

$$T_{moy} = \frac{n^2 + 1}{2}$$

### Conclusion
The average case complexity formula is strictly $\frac{n^2 + 1}{2}$. 
If we were to convert this to Big-O notation, we drop the constants and divisors, revealing an average case complexity of $\Theta(n^2)$.
