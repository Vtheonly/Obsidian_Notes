# 4. Mathematical Proofs of Big O Notation

> [!info] The Formal Definition of Big-O
> You cannot just guess if $f(n) \in O(g(n))$. You must prove it using the formal mathematical definition:
> $f(n) \in O(g(n)) \iff \exists c > 0, n_0 > 0 : f(n) \le c \cdot g(n)$ for all $n \ge n_0$.
> 
> **In simple terms:** You must find a constant number $c$ that, when multiplied by $g(n)$, makes $g(n)$ forever larger than or equal to $f(n)$ after a certain point ($n_0$).

## Problem 1: Prove $2^{n+1} = O(2^n)$

**Goal:** Determine if this statement is true or false.

1. **Identify functions:**
   $f(n) = 2^{n+1}$
   $g(n) = 2^n$

2. **Set up the inequality:**
   $f(n) \le c \cdot g(n)$
   $2^{n+1} \le c \cdot 2^n$

3. **Algebraic Manipulation:**
   Recall the exponent rule: $x^{a+b} = x^a \cdot x^b$.
   We can split $2^{n+1}$ into $2^n \times 2^1$.

   Substitute this back into the inequality:
   $2^n \cdot 2^1 \le c \cdot 2^n$
   $2^n \cdot 2 \le c \cdot 2^n$

4. **Isolate $c$:**
   Because $2^n$ is positive for all $n$, we can divide both sides by $2^n$:
   $2 \le c$

5. **Conclusion:**
   This inequality holds perfectly true. If we pick any constant $c \ge 2$, the condition is satisfied for all $n \ge 1$.
   Because we successfully found a valid constant $c$, the formula is **Vrai (True)**.

## Problem 2: Prove $2^{2n} = O(2^n)$

**Goal:** Determine if this statement is true or false.

1. **Identify functions:**
   $f(n) = 2^{2n}$
   $g(n) = 2^n$

2. **Set up the inequality:**
   $f(n) \le c \cdot g(n)$
   $2^{2n} \le c \cdot 2^n$

3. **Algebraic Manipulation:**
   Recall the exponent rule: $x^{ab} = (x^a)^b$.
   We can rewrite $2^{2n}$ as $(2^n)^2$.

   Substitute this back into the inequality:
   $(2^n)^2 \le c \cdot 2^n$

4. **Isolate $c$:**
   Divide both sides by $2^n$ (which is legal since $2^n > 0$):
   $2^n \le c$

5. **Conclusion:**
   Look closely at the final inequality: $2^n \le c$. 
   We are trying to find a *constant* number $c$ (like 10, 100, or 1000) that is always bigger than $2^n$ for *all* $n$ going to infinity. 
   This is mathematically impossible. As $n$ grows, $2^n$ grows infinitely large and will eventually surpass any fixed constant $c$. 
   Because no such constant $c$ can exist, the condition fails. The formula is **Faux (False)**.

> [!tip] Common Mistake
> Students often assume $2^{2n}$ is just "twice as much" as $2^n$. It is not!
> $2^{2n} = (2^n)^2 = 4^n$. 
> $4^n$ grows exponentially faster than $2^n$. Therefore, $4^n$ can never be bounded by $O(2^n)$.
