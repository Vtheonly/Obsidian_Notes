# 2. Asymptotic Notations and Big-O Proofs

To evaluate the efficiency of an algorithm, we look at how the number of operations grows as the input size $n$ approaches infinity. This is known as asymptotic analysis.

## Core Definitions

> [!info] The Formal Definition of Big-O
> Big-O notation, $f(n) \in O(g(n))$, signifies that $f(n)$ is **upper-bounded** by $g(n)$ multiplied by some constant. 
> Mathematically: 
> $\exists c > 0, \exists n_0 > 0 \text{ such that } \forall n \ge n_0, f(n) \le c \cdot g(n)$

We also have:
* **Big-Omega ($\Omega$)**: Lower bound. $f(n) \ge c \cdot g(n)$.
* **Big-Theta ($\Theta$)**: Tight bound. Exists bounded by both $c_1 \cdot g(n)$ and $c_2 \cdot g(n)$.

## Exercise 4: Exponential Bounds Proofs
Are the following formulas correct? Justify using the formal definition.

### Problem 1: $2^{n+1} = O(2^n)$
**Hypothesis:** Is $f(n) = 2^{n+1}$ bounded by $g(n) = 2^n$?
Let's use basic exponent rules: 
$$ 2^{n+1} = 2^1 \times 2^n = 2 \cdot 2^n $$

According to the Big-O definition, we need to find a constant $c$ such that:
$$ 2 \cdot 2^n \le c \cdot 2^n $$

If we choose $c = 2$, the inequality becomes $2 \cdot 2^n \le 2 \cdot 2^n$, which is perfectly true for all $n \ge 1$ (where $n_0 = 1$).
**Conclusion:** **TRUE**. $2^{n+1}$ is indeed $O(2^n)$.

### Problem 2: $2^{2n} = O(2^n)$
**Hypothesis:** Is $f(n) = 2^{2n}$ bounded by $g(n) = 2^n$?
Let's rewrite $2^{2n}$ using exponent rules:
$$ 2^{2n} = (2^2)^n = 4^n $$

Now, try to satisfy the Big-O inequality:
$$ 4^n \le c \cdot 2^n $$
Divide both sides by $2^n$:
$$ \frac{4^n}{2^n} \le c \implies 2^n \le c $$

Is there a constant $c$ that is always greater than $2^n$ as $n$ goes to infinity? **No.** Because $2^n$ grows infinitely large, no static constant $c$ can bound it.
**Conclusion:** **FALSE**. $2^{2n}$ is NOT $O(2^n)$.

## Exercise 5: Comparing Function Pairs
For the following pairs, indicate if $f \in O(g)$, $g \in O(f)$, or both.

### (a) $f(n) = 100n, \quad g(n) = 0.1 \cdot n^2$
*   $f(n)$ is a linear function $O(n)$.
*   $g(n)$ is a quadratic function $O(n^2)$.
*   Constants ($100$ and $0.1$) do not matter asymptotically. As $n \to \infty$, $n^2$ will always dwarf $n$.
*   **Result:** $f \in O(g)$

### (b) $f(n) = \sqrt[3]{n^2}, \quad g(n) = \sqrt{n}$
*   Rewrite with fractional exponents:
    *   $f(n) = n^{2/3} \approx n^{0.66}$
    *   $g(n) = n^{1/2} = n^{0.5}$
*   Since $0.66 > 0.5$, $f(n)$ grows faster than $g(n)$.
*   **Result:** $g \in O(f)$

### (c) $f(n) = \log_2(2^n \cdot n^3), \quad g(n) = 3n$
*   First, simplify $f(n)$ using logarithm properties ($\log(A \cdot B) = \log A + \log B$ and $\log(x^y) = y \log x$):
    $$ f(n) = \log_2(2^n) + \log_2(n^3) = n + 3\log_2(n) $$
*   Asymptotically, $n$ grows much faster than $3\log_2(n)$. Therefore, we drop the lower-order term. $f(n) \approx O(n)$.
*   $g(n) = 3n$, which is also $O(n)$.
*   **Result:** They bound each other. $f \in O(g)$ AND $g \in O(f)$ (which implies $f \in \Theta(g)$).
