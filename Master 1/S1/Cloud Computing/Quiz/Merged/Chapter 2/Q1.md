>> [!success]- Answer
>> False

> [!question] Containers are lightweight virtual runtimes that package application code alongside its dependencies, enabling them to deploy faster than standard virtual machines.
>> [!success]- Answer
>> True

> [!question] In a Platform as a Service (PaaS) deployment model, the developer is responsible for patching and upgrading the host operating system.
>> [!success]- Answer
>> False

> [!question] The Runtime Cloud Layer is a virtual execution environment that coordinates CPU, RAM, virtual storage, and network allocations.
>> [!success]- Answer
>> True

> [!question] Public cloud models eliminate the risk of performance fluctuations from "noisy neighbors" because physical hardware is never shared between multiple tenants.
>> [!success]- Answer
>> False

> [!question] A community cloud is built to be dedicated exclusively to a single organization and cannot be managed by a third-party provider.
>> [!success]- Answer
>> False

> [!question] Software as a Service (SaaS) eliminates the need for local software installations because the applications run directly inside a web browser.
>> [!success]- Answer
>> True

> [!question] In the Infrastructure as a Service (IaaS) model, the cloud provider manages the virtualization and hypervisor software layer.
>> [!success]- Answer
>> True

>> [!success]- Answer
>> True

> [!question] Under the Shared Responsibility Model, Access Rights & Identities (IAM) remain the responsibility of the customer across all cloud service models (IaaS, PaaS, and SaaS).

### Part 2: Multiple Choice Questions

> [!question] Which architectural layer acts as an orchestrating bridge, linking applications, operating systems, and basic cloud assets together?
> a) Infrastructure Layer
> b) Management Layer (Middleware)
> c) Application Layer
> d) Security Control Plane
>> [!success]- Answer
>> b) Management Layer (Middleware)

> [!question] What is the primary target audience of the Platform as a Service (PaaS) model?
> a) End Users
> b) System & Network Architects
> c) Application Developers
> d) Facility Security Guards
>> [!success]- Answer
>> c) Application Developers

> [!question] If an attacker exploits an unpatched guest operating system vulnerability inside an IaaS virtual machine, who is responsible for the breach?
> a) The Cloud Provider
> b) The Customer
> c) The Hypervisor Vendor
> d) Both parties equally
>> [!success]- Answer
>> b) The Customer

> [!question] Which cloud deployment model integrates private resources with public cloud services, allowing workloads to scale dynamically between them?
> a) Hybrid Cloud
> b) Private Cloud
> c) Public Cloud
> d) Community Cloud
>> [!success]- Answer
>> a) Hybrid Cloud

> [!question] Which of the following is an industry example of an Infrastructure as a Service (IaaS) solution?
> a) Heroku
> b) Google App Engine
> c) Amazon EC2
> d) Google Workspace
>> [!success]- Answer
>> c) Amazon EC2

> [!question] What virtualization component is installed directly on physical host systems to split hardware resources into isolated virtual environments?
> a) Cloud Management Platform
> b) Hypervisor (Virtual Machine Monitor)
> c) System Image
> d) Cluster Nodes
>> [!success]- Answer
>> b) Hypervisor (Virtual Machine Monitor)

> [!question] Under the Shared Responsibility Model, who is responsible for application code security and database logic within a PaaS environment?
> a) The Cloud Provider
> b) The Hardware Manufacturer
> c) The Customer
> d) The Middleware Vendor
>> [!success]- Answer
>> c) The Customer

> [!question] Which deployment model is defined by sharing infrastructure costs among a specific group of participating organizations with shared compliance or research goals?
> a) Private Cloud
> b) Public Cloud
> c) Community Cloud
> d) Multi-Tenant Hybrid Cloud
>> [!success]- Answer
>> c) Community Cloud

> [!question] Which of the following is a noted disadvantage of Software as a Service (SaaS)?
> a) Complex operating system management
> b) Minimal customization options for source code or databases
> c) High upfront physical CAPEX costs
> d) The requirement to configure physical network switches
>> [!success]- Answer
>> b) Minimal customization options for source code or databases

> [!question] In which cloud service model does the provider handle physical hardware, operating systems, middlewares, and runtimes, leaving the customer responsible only for application code and configurations?
> a) Infrastructure as a Service (IaaS)
> b) Platform as a Service (PaaS)
> c) Software as a Service (SaaS)
> d) On-Premise
>> [!success]- Answer
>> b) Platform as a Service (PaaS)
