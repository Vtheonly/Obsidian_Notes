### Part 2: Multiple Choice Questions

> [!question] Which backend component provides identity and access management (IAM), encryption, firewalls, and data backups?
> a) Management Layer (Middleware)
> b) Security Control Plane
> c) Service Layer
> d) Runtime Cloud Layer
>> [!success]- Answer
>> b) Security Control Plane

> [!question] Which of the following is a key advantage of Platform as a Service (PaaS)?
> a) Direct physical network configuration
> b) Zero operating system and middleware management for developers
> c) Full root-level control over the physical host machine
> d) Custom hypervisor installation
>> [!success]- Answer
>> b) Zero operating system and middleware management for developers

> [!question] What is the term for preconfigured OS files deployed instantly to create uniform virtual environments?
> a) Hypervisors
> b) Nodes
> c) System Images (OS Templates)
> d) Containers
>> [!success]- Answer
>> c) System Images (OS Templates)

> [!question] Which deployment model is best suited for a financial institution with strict compliance requirements and predictable, stable workloads?
> a) Public Cloud
> b) Private Cloud
> c) Community Cloud
> d) Dynamic Bursting Cloud
>> [!success]- Answer
>> b) Private Cloud

> [!question] What is a primary risk for an organization that shares a confidential document using a publicly accessible link on a SaaS platform?
> a) Hardware failure of the SaaS server
> b) Compromise of the hypervisor logical isolation
> c) Data leakage due to poor customer access management
> d) Inability to patch the SaaS host operating system
>> [!success]- Answer
>> c) Data leakage due to poor customer access management

> [!question] Which layer of the cloud architecture manages physical hardware resources located in datacenters alongside basic virtualization programs?
> a) Application Layer
> b) Service Layer
> c) Infrastructure Layer
> d) Runtime Cloud Layer
>> [!success]- Answer
>> c) Infrastructure Layer

> [!question] In the case study of Netflix, why is AWS EC2 (IaaS) highly beneficial?
> a) It allows Netflix to avoid managing operating systems completely.
> b) It provides a complete development environment with pre-installed databases.
> c) It allows renting and scaling virtual instances up or down based on peak hour demand.
> d) It forces developers to run code via a pre-configured web browser interface.
>> [!success]- Answer
>> c) It allows renting and scaling virtual instances up or down based on peak hour demand.

> [!question] Which layer of the backend architecture contains the software applications that end-users directly interact with to perform tasks?
> a) Service Layer
> b) Application Layer
> c) Storage Layer
> d) Runtime Cloud Layer
>> [!success]- Answer
>> b) Application Layer

> [!question] Who is responsible for database engine configuration and support under the Platform as a Service (PaaS) model?
> a) The Customer
> b) The Provider
> c) The Database Administrator hired by the client
> d) Both parties share it equally
>> [!success]- Answer
>> b) The Provider

> [!question] What is an advantage of a Community Cloud deployment model?
> a) Maximum customization over physical server design
> b) Lower upfront cost compared to a fully dedicated Private Cloud due to cost-sharing
> c) Infinite resource elasticity over the entire public internet
> d) Completely independent governance with zero cooperation required
>> [!success]- Answer
>> b) Lower upfront cost compared to a fully dedicated Private Cloud due to cost-sharing
