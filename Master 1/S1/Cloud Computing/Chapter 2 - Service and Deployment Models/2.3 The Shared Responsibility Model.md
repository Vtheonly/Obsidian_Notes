### 2.3.2. Essential Distinction: Security OF vs. Security IN the Cloud
*   **Security OF the Cloud (Provider's Duty):** The provider is responsible for securing the infrastructure that runs all services. This includes physical datacenters, host servers, network switching fabrics, power grids, and hypervisors.
*   **Security IN the Cloud (Customer's Duty):** The customer is responsible for protecting whatever they deploy inside the cloud. This includes configuring firewalls (Security Groups), patching virtual operating systems, implementing multi-factor authentication (MFA) for user accounts, and managing application vulnerabilities.
