Build a complete command-line modular data processing pipeline using the following structure:
```

├── data/
│   ├── raw/
```

#### Tasks
1. **`utils/helpers.py`**:
   * Implement `clean_lines(lines)`: Takes a list of strings, trims trailing/leading spaces, removes empty elements, and returns a cleaned list of records.
   * Implement `parse_products(lines)`: Converts a list of lines into tuples of `(name, price, quantity)`. Ignores malformed records and converts metrics to numeric types.
2. **`utils/product.py`**:
   * Implement the `Product` class with a `total_value()` method that returns the product's total inventory value (`price * quantity`).
3. **`scripts/create_products.py`**:
   * Load raw sales data from `data/raw/data.txt`, process it using your helper utilities, instantiate `Product` objects, and compute total sales metrics.
4. **`notebooks/analysis.ipynb`**:
   * Read the data, generate the object instances, and visualize total sales by product using a Matplotlib bar chart.

### Step-by-Step Implementation Guide

#### 1. Implement Utility Package Files

##### Creating `utils/helpers.py`
```python
# utils/helpers.py

def clean_lines(lines: list) -> list:
    """
    Cleans a list of strings by stripping leading/trailing whitespace,
    removing newline characters, and filtering out empty elements.
    """
    cleaned_list = []
    for line in lines:
        stripped_line = line.strip()
        # Filter out empty strings and newlines
        if stripped_line != "" and stripped_line != "\n":
            cleaned_list.append(stripped_line)
    return cleaned_list


def parse_products(lines: list) -> list:
    """
    Parses cleaned lines into structured product tuples (name, price, quantity).
    Gracefully ignores and skips malformed records.
    """
    parsed_records = []
    for line in lines:
        parts = line.split(",")
        # Check if the row contains exactly three values (Name, Price, Quantity)
        if len(parts) != 3:
            print(f"[Warning] Skipping malformed record (incorrect field count): {line}")
            continue
        try:
            name = parts[0].strip()
            price = float(parts[1].strip())
            quantity = int(parts[2].strip())
            parsed_records.append((name, price, quantity))
        except ValueError as e:
            # Handle non-numeric conversion failures
            print(f"[Warning] Skipping malformed record (conversion error): {line}. Error details: {e}")
            continue
    return parsed_records
```

##### Creating `utils/product.py`
```python
# utils/product.py

class Product:
    def __init__(self, name: str, price: float, quantity: int):
        self.name = name
        self.price = price
        self.quantity = quantity
        
    def total_value(self) -> float:
        """
        Calculates the total inventory value for this product.
        """
        return self.price * self.quantity
        
    def __str__(self) -> str:
        return f"Product(Name: {self.name}, Price: {self.price}, Quantity: {self.quantity})"
```

##### Creating Package Initialization File `utils/__init__.py`

#### 3. Implement Execution Orchestrator Script
Create `scripts/create_products.py`:

```python
# scripts/create_products.py

import sys
import os

# Append project root to system path to enable modular absolute importing
sys.path.append(os.path.abspath(os.path.join(os.path.dirname(__file__), '..')))

from utils import clean_lines, parse_products, Product

def main():
    raw_data_path = "data/raw/data.txt"
    
    # 1. Read raw file contents
    if not os.path.exists(raw_data_path):
        print(f"[Error] Target raw data path not found: {raw_data_path}")
        sys.exit(1)
        
    with open(raw_data_path, "r") as file:
        raw_lines = file.readlines()
        
    # 2. Apply cleaning and parsing helpers
    cleaned_lines = clean_lines(raw_lines)
    parsed_tuples = parse_products(cleaned_lines)
    
    # 3. Instantiate objects and compute metrics
    product_objects = []
    overall_total_sales = 0.0
    
    print("\n--- Processing Product Inventories ---")
    for name, price, qty in parsed_tuples:
        prod_obj = Product(name, price, qty)
        product_objects.append(prod_obj)
        
        inventory_value = prod_obj.total_value()
        overall_total_sales += inventory_value
        print(f"{prod_obj.name}: {inventory_value:.1f}")
        
    print("--------------------------------------")
    print(f"Total Sales: {overall_total_sales:.1f}")

if __name__ == "__main__":
    main()
```
