## 3.1. Python Essentials and Data Structures

### 1. Variables and Dynamic Type Conversion
Python is a dynamically-typed, garbage-collected language. Variable types are determined at runtime rather than compile-time. Explicit type conversion (type casting) is essential when reading data from external sources (which are often imported as strings).

```python
# Variable definitions
age_str = "45"
bp_str = "134.5"

# Type casting conversions
age_int = int(age_str)         # String to integer conversion
bp_float = float(bp_str)       # String to floating point conversion
is_hypertensive = bool(bp_float > 130.0) # Condition evaluates to Boolean True

print(f"Age: {type(age_int)}, BP: {type(bp_float)}, High BP Status: {type(is_hypertensive)}")
```

### 2. Operators Cheat Sheet
* **Arithmetic**: `+`, `-`, `*`, `/` (float division), `//` (floor division), `%` (modulo), `**` (exponentiation).
* **Comparison**: `==`, `!=`, `<`, `>`, `<=`, `>=`.
* **Logical**: `and`, `or`, `not`.

### 3. Data Structures Comparison

| Data Structure | Syntax Example | Mutable | Ordered | Allows Duplicates | Big-O Access | Primary Use Case |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| **List** | `[1, 2, 2, 3]` | Yes | Yes | Yes | $O(n)$ search, $O(1)$ index access | Dynamic, sequential arrays. |
| **Tuple** | `(1, 2, 2, 3)` | No | Yes | Yes | $O(n)$ search, $O(1)$ index access | Read-only records, database rows. |
| **Dictionary** | `{"id": 1, "age": 25}` | Yes | Yes (3.7+) | No keys, yes values | $O(1)$ average search and insertion | Key-value mapping, structured metadata. |
| **Set** | `{1, 2, 3}` | Yes | No | No | $O(1)$ average membership check | Deduplication, mathematical set operations. |
