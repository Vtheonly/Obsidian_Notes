

## 5.4. Feature Scaling and Normalization

Many machine learning algorithms (like KNN, SVM, and gradient descent-based models) are sensitive to the scale of input features. Scaling ensures all features contribute equally to the model's distance calculations.

### Mathematical Scaling Formulas

| Scaling Method | Formula | Primary Characteristics | Common Use Cases |
| :--- | :--- | :--- | :--- |
| **Min-Max Scaling** | $$x' = \frac{x - x_{\min}}{x_{\max} - x_{\min}}$$ | Rescales features to a fixed range, usually $[0, 1]$. Highly sensitive to outliers. | Neural Networks, algorithms requiring positive bounds. |
| **Standardization (Z-Score)** | $$x' = \frac{x - \mu}{\sigma}$$ | Rescales features to have a mean of 0 and a standard deviation of 1. | Linear/Logistic Regression, PCA, Support Vector Machines. |
| **Robust Scaling** | $$x' = \frac{x - \text{median}(x)}{\text{IQR}(x)}$$ | Scales features using the median and IQR, making it highly robust to outliers. | Datasets with extreme outliers. |
| **L1 Normalization** | $$x' = \frac{x}{\|x\|_1} = \frac{x}{\sum |x_i|}$$ | Normalizes feature vectors such that the sum of their absolute values equals 1. | Text mining (Bag-of-Words, TF-IDF). |
| **L2 Normalization** | $$x' = \frac{x}{\|x\|_2} = \frac{x}{\sqrt{\sum x_i^2}}$$ | Normalizes feature vectors such that their Euclidean length equals 1. | Cosine similarity calculations, KNN, clustering. |
