

## 5.5. Tutorial 5. Medical Dataset Preprocessing Walkthrough

### Problem Statement
You are given a messy medical dataset with missing values, inconsistent text formatting, and unscaled columns:

| PatientID | Age | BloodPressure | Cholesterol | Gender | Diagnosis |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **P001** | 25 | 120 | 200 | Female | Diabetes |
| **P002** | NaN | 130 | 180 | male | Healthy |
| **P003** | 65 | NaN | 250 | Male | Diabetes |
| **P004** | 50 | 110 | 190 | female | Healthy |
| **P005** | 80 | 200 | 300 | MALE | Hypertension |
| **P006** | 120 | 95 | 180 | None | Healthy |
| **P007** | 45 | 125 | NaN | female | Diabetes |
| **P008** | 38 | 118 | 210 | female | Healthy |
| **P009** | NaN | 122 | 220 | FEMALE | None |
| **P010** | 55 | 210 | 350 | Male | Hypertension |

Perform the following preprocessing tasks:
1. **Clean Categorical Values**: Standardize spelling and formatting in `Gender` and `Diagnosis`.
2. **Handle Missing Categorical Values**: Impute missing categorical values with the mode.
3. **Handle Missing Numerical Values**: Impute missing values in `Age`, `BloodPressure`, and `Cholesterol` using median imputation.
4. **Outlier Mitigation**: Identify and cap extreme outliers using the IQR method.
5. **Categorical Encoding**: One-hot encode the standardized categorical features.
6. **Feature Scaling**: Apply standard scaling to numerical columns.

---

### Step-by-Step Implementation

```python
import pandas as pd
import numpy as np
from sklearn.preprocessing import StandardScaler

# 1. Initialize the dataset
data = {
    'PatientID': ['P001', 'P002', 'P003', 'P004', 'P005', 'P006', 'P007', 'P008', 'P009', 'P010'],
    'Age': [25, np.nan, 65, 50, 80, 120, 45, 38, np.nan, 55],
    'BloodPressure': [120, 130, np.nan, 110, 200, 95, 125, 118, 122, 210],
    'Cholesterol': [200, 180, 250, 190, 300, 180, np.nan, 210, 220, 350],
    'Gender': ['Female', 'male', 'Male', 'female', 'MALE', None, 'female', 'female', 'FEMALE', 'Male'],
    'Diagnosis': ['Diabetes', 'Healthy', 'Diabetes', 'Healthy', 'Hypertension', 'Healthy', 'Diabetes', 'Healthy', None, 'Hypertension']
}
df = pd.DataFrame(data)

# 2. Clean and Standardize Categorical Features
df['Gender'] = df['Gender'].astype(str).str.lower().str.strip()
df['Gender'] = df['Gender'].replace('none', np.nan) # Map string 'none' to actual NaN

df['Diagnosis'] = df['Diagnosis'].astype(str).str.lower().str.strip()
df['Diagnosis'] = df['Diagnosis'].replace('none', np.nan)

# 3. Impute Missing Categorical Features with the Mode
gender_mode = df['Gender'].mode()[0]
df['Gender'] = df['Gender'].fillna(gender_mode)

diagnosis_mode = df['Diagnosis'].mode()[0]
df['Diagnosis'] = df['Diagnosis'].fillna(diagnosis_mode)

# 4. Impute Missing Numerical Features with the Median
numerical_cols = ['Age', 'BloodPressure', 'Cholesterol']
for col in numerical_cols:
    median_val = df[col].median()
    df[col] = df[col].fillna(median_val)

# 5. Identify and Cap Outliers using the IQR Method
for col in numerical_cols:
    Q1 = df[col].quantile(0.25)
    Q3 = df[col].quantile(0.75)
    IQR = Q3 - Q1
    lower_bound = Q1 - 1.5 * IQR
    upper_bound = Q3 + 1.5 * IQR
    
    # Cap values outside the IQR boundaries
    df[col] = np.clip(df[col], lower_bound, upper_bound)

# 6. One-Hot Encode Categorical Features
# Drop the first category to prevent the dummy variable trap
df_encoded = pd.get_dummies(df, columns=['Gender', 'Diagnosis'], drop_first=True, dtype=int)

# 7. Scale Numerical Features using Z-Score Standardization
scaler = StandardScaler()
df_encoded[numerical_cols] = scaler.fit_transform(df_encoded[numerical_cols])

# Output the fully processed dataset
print("--- Preprocessed Medical Dataset ---")
print(df_encoded.to_string())
```

##### Output Verification
* Categorical values are standardized (e.g., `female`, `male`, `diabetes`, `healthy`).
* Missing entries in `Age`, `BloodPressure`, and `Cholesterol` are replaced with their respective medians.
* Outliers (like Age 120) are capped within the IQR boundaries.
* Categorical columns are converted to binary one-hot encoded columns (e.g., `Gender_male`, `Diagnosis_healthy`).
* Numerical features are normalized to a mean of 0 and a standard deviation of 1.
