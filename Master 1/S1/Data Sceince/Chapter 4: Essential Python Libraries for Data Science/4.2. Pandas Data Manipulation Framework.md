

## 4.2. Pandas Data Manipulation Framework

Pandas provides structured, high-performance data structures—**Series** (1D) and **DataFrames** (2D)—designed for tabular data analysis.

### 1. Core Indexes: `.loc[]` vs. `.iloc[]`
* **`.loc[]`**: Label-based indexing. Accesses rows and columns using their string labels (includes the stop index in slices).
* **`.iloc[]`**: Integer-based indexing. Accesses positions using integer coordinates (excludes the stop index, matching standard Python slicing).

```python
import pandas as pd

df = pd.DataFrame(
    data=[[10, "A"], [20, "B"], [30, "C"]],
    index=["row1", "row2", "row3"],
    columns=["Metric", "Category"]
)

# Access using .loc (label)
print(df.loc["row1":"row2", "Metric"]) # Returns rows 'row1' and 'row2'

# Access using .iloc (positional)
print(df.iloc[0:2, 0]) # Returns index positions 0 and 1
```

### 2. Complex Row Filtering and Querying
Filter rows using logical operators (`&` for AND, `|` for OR, `~` for NOT):

```python
# Filtering patients over 50 with hypertension
high_risk = df[(df['Age'] > 50) & (df['Diagnosis'] == 'Hypertension')]

# Equivalent alternative using the readable query syntax
high_risk_query = df.query("Age > 50 and Diagnosis == 'Hypertension'")
```

### 3. Reshaping Frameworks: Pivot, Pivot Table, and Melt
* **`pivot()`**: Reshapes data from long format to wide format. Requires unique index-column pairs (does not perform aggregations).
* **`pivot_table()`**: Reshapes data like `pivot()`, but can aggregate duplicate entries (e.g., computing the mean, sum, or count).
* **`melt()`**: Unpivots a DataFrame from wide format to long format.
