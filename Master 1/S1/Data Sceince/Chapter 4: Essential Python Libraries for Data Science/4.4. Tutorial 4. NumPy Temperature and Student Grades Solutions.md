
## 4.4. Tutorial 4. NumPy Temperature and Student Grades Solutions

### Problem Statement

#### Exercise 1: Temperature Analysis with NumPy
You are given a table of daily temperatures (°C) measured over 7 days in 3 cities (Alger, Annaba, Oran):

| Day | Alger | Annaba | Oran |
| :--- | :--- | :--- | :--- |
| **1** | 22 | 25 | 20 |
| **2** | 24 | 27 | 21 |
| **3** | 19 | 20 | 22 |
| **4** | 25 | 29 | 28 |
| **5** | 26 | 30 | 27 |
| **6** | 21 | 21 | 23 |
| **7** | 20 | 26 | 25 |

Implement the following tasks:
1. Construct a 2D NumPy array `T` containing the temperature values.
2. Calculate and display:
   * The maximum temperature recorded for each city.
   * The maximum temperature recorded for each day.
   * The overall maximum temperature in the dataset.
3. Identify the exact day and city where the overall maximum temperature occurred.
4. Create a new array showing the highest temperature between Alger and Oran for each day.
5. Append a new row to the array representing the average temperature of each city over the 7 days.
6. Identify the days when Annaba was the hottest of the three cities.

---

#### Exercise 2: Student Grades Analysis with NumPy
Create a dataset representing the grades of 100 students in 5 modules (THL, DSS, DB, SE, RE):
1. Generate integer grades between 0 and 20 using `np.random.randint`, with seed `101` for reproducibility.
2. Compute basic statistical summaries: Mean, median, variance, and standard deviation per module, along with overall dataset limits.
3. Compute percentiles (25th, 50th, 75th) for the THL module.
4. Use boolean masking to find and extract all grades between 10 and 15.
5. Create a Matplotlib visualization showing the average grades across modules and their distributions.

---

### Complete Code Solutions

#### Exercise 1: NumPy Temperature Code Solution
```python
import numpy as np

# 1. Create temperature matrix T
T = np.array([
    [22, 25, 20],
    [24, 27, 21],
    [19, 20, 22],
    [25, 29, 28],
    [26, 30, 27],
    [21, 21, 23],
    [20, 26, 25]
])

# Define labels for mapping indices to real-world names
cities = ["Alger", "Annaba", "Oran"]
days = [f"Day {i}" for i in range(1, 8)]

# 2. Maximum Calculations
max_per_city = T.max(axis=0)  # Maximum across rows (for each column/city)
max_per_day = T.max(axis=1)   # Maximum across columns (for each row/day)
overall_max = T.max()

print("Maximum temperature per city:")
for city, temp in zip(cities, max_per_city):
    print(f"  {city}: {temp}°C")

print("\nMaximum temperature per day:")
for day, temp in zip(days, max_per_day):
    print(f"  {day}: {temp}°C")

print(f"\nOverall Maximum Temperature: {overall_max}°C")

# 3. Find index position of overall maximum
max_index = T.argmax()  # Flattened index
row_idx, col_idx = np.unravel_index(max_index, T.shape) # Unravel to 2D coordinates
print(f"Overall maximum occurred on {days[row_idx]} in {cities[col_idx]} ({overall_max}°C).")

# 4. Compare Alger (col 0) and Oran (col 2) using np.maximum
alger_vs_oran = np.maximum(T[:, 0], T[:, 2])
print("\nHighest temperature between Alger and Oran for each day:")
for day, temp in zip(days, alger_vs_oran):
    print(f"  {day}: {temp}°C")

# 5. Append a row with the average temperature of each city
city_averages = T.mean(axis=0).reshape(1, 3) # Calculate average and reshape to 1x3 row vector
T_with_average = np.vstack([T, city_averages]) # Stack vertically
print("\nTemperature matrix with appended averages (Row 8):")
print(np.round(T_with_average, 2))

# 6. Identify the days when Annaba was the hottest of the three cities
# Annaba is column index 1 (Alger is 0, Oran is 2)
annaba_hottest_mask = (T[:, 1] > T[:, 0]) & (T[:, 1] > T[:, 2])
print("\nDays when Annaba was the hottest city:")
for idx, is_hottest in enumerate(annaba_hottest_mask):
    if is_hottest:
        print(f"  {days[idx]} (Annaba temperature: {T[idx, 1]}°C)")
```

#### Exercise 2: NumPy Student Grades Code Solution
```python
import numpy as np
import matplotlib.pyplot as plt

# 1. Generate student grades dataset
np.random.seed(101) # Set seed for reproducibility
grades_of_students = np.random.randint(0, 21, size=(100, 5)) # 100 rows (students), 5 columns (modules)
modules = ["THL", "DSS", "DB", "SE", "RE"]

# 2. Compute basic statistical summaries
print("--- Module-Level Statistical Summaries ---")
for idx, module in enumerate(modules):
    mod_grades = grades_of_students[:, idx]
    print(f"Module {module}:")
    print(f"  Mean: {mod_grades.mean():.2f}")
    print(f"  Median: {np.median(mod_grades):.2f}")
    print(f"  Variance: {mod_grades.var():.2f}")
    print(f"  Std Dev: {mod_grades.std():.2f}")

# Overall boundaries
print("\nDataset Boundaries:")
print(f"  Minimum Grade: {grades_of_students.min()}")
print(f"  Maximum Grade: {grades_of_students.max()}")

# 3. Compute percentiles for the THL module (index 0)
thl_grades = grades_of_students[:, 0]
p25 = np.percentile(thl_grades, 25)
p50 = np.percentile(thl_grades, 50)
p75 = np.percentile(thl_grades, 75)
print(f"\nTHL Percentiles (25th, 50th, 75th): {p25:.1f}, {p50:.1f}, {p75:.1f}")

# 4. Filter grades between 10 and 15 (inclusive) using boolean masking
grades_mask = (grades_of_students >= 10) & (grades_of_students <= 15)
filtered_grades = grades_of_students[grades_mask]
print(f"\nNumber of grades between 10 and 15: {len(filtered_grades)}")
print(f"First 15 filtered grades: {filtered_grades[:15]}")

# 5. Visualizations
fig, axs = plt.subplots(1, 2, figsize=(14, 5))

# Plot 1: Bar chart comparing average grades per module
avg_grades = grades_of_students.mean(axis=0)
axs[0].bar(modules, avg_grades, color='skyblue', edgecolor='black', alpha=0.8)
axs[0].set_title("Average Grade per Module", fontsize=12, fontweight='bold')
axs[0].set_xlabel("Modules")
axs[0].set_ylabel("Average Score (Out of 20)")
axs[0].set_ylim(0, 20)
axs[0].grid(True, axis='y', linestyle='--', alpha=0.6)

# Plot 2: Boxplot showing grade distributions per module
axs[1].boxplot(grades_of_students, labels=modules, patch_artist=True,
                boxprops=dict(facecolor='lightgreen', color='black'),
                medianprops=dict(color='darkred', linewidth=2))
axs[1].set_title("Grade Distributions per Module", fontsize=12, fontweight='bold')
axs[1].set_xlabel("Modules")
axs[1].set_ylabel("Grade Range")
axs[1].grid(True, axis='y', linestyle='--', alpha=0.4)

plt.tight_layout()
plt.show()
```
