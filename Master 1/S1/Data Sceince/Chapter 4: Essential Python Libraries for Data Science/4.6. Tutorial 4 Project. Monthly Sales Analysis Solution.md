
## 4.6. Tutorial 4 Project. Monthly Sales Analysis Solution

### Problem Statement
Generate, analyze, and visualize monthly sales data for four products over one year.

```
project_sales/
├── notebook.ipynb          # Notebook for analysis and plotting
├── utils.py                # Functions for data generation
└── data/
    ├── initial.csv         # Raw generated dataset
    ├── final.csv           # Cleaned dataset with computed metrics
    └── output.csv          # Final summary table (pivot tables)
```

#### Step 1: Data Generation
1. Generate monthly dates for one year (e.g., `'2025-01-01'` to `'2025-12-01'`).
2. Build a generator function in `utils.py` that returns a random NumPy array of integers for the given size.
3. Generate monthly units sold for four products within these target ranges:
   * **Product A**: 50–100 units
   * **Product B**: 30–80 units
   * **Product C**: 20–60 units
   * **Product D**: 10–50 units
4. Combine this data into a DataFrame with columns: `Date`, `Product_A`, `Product_B`, `Product_C`, `Product_D`.
5. Export this raw dataset as `data/initial.csv`.

#### Step 2: Build Computed Metrics DataFrame
1. Load `data/initial.csv`.
2. Compute the following metrics for each month:
   * `Total_Sales`: Sum of all units sold across all products.
   * `Average_Sales`: Mean units sold across all products.
   * `Month_over_Month_Growth`: Percent change in `Total_Sales` compared to the previous month.
3. Assign each month to a quarter:
   * **Q1**: Jan–Mar, **Q2**: Apr–Jun, **Q3**: Jul–Sep, **Q4**: Oct–Dec.
4. Add reference columns:
   * `Max_Sales_Product`: The product with the highest sales in that month.
   * `Min_Sales_Product`: The product with the lowest sales in that month.
5. Export the enriched DataFrame to `data/final.csv`.

#### Step 3: Pivot Tables and Summary Aggregations
1. Create a pivot table summarizing average sales by quarter for each product.
2. Export this summary to `data/output.csv`.

#### Step 4: Extract Key Insights
Identify the best month (highest total sales), best product (highest cumulative annual sales), and best quarter (highest total sales).

#### Step 5: Visualizations
Generate a line chart of product sales trends, a stacked bar chart of monthly sales, a monthly sales heatmap, and a boxplot showing product sales distributions.

### Complete Code Implementation

#### 1. Implement Utility Module `utils.py`
```python
# utils.py
import numpy as np

def generate_random_sales(min_val: int, max_val: int, size: int) -> np.ndarray:
    """
    Generates a 1D NumPy array of random integers within a specified range.
    """
    return np.random.randint(low=min_val, high=max_val + 1, size=size)
```

---

#### 2. Implement Main Execution Notebook or Script

```python
import os
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns
from utils import generate_random_sales

# Ensure directory structure exists
os.makedirs("data", exist_ok=True)

# --- 1. Data Generation ---
np.random.seed(42) # Set seed for reproducibility
months = pd.date_range(start="2025-01-01", end="2025-12-01", freq="MS")
num_months = len(months)

sales_data = {
    "Date": months,
    "Product_A": generate_random_sales(50, 100, num_months),
    "Product_B": generate_random_sales(30, 80, num_months),
    "Product_C": generate_random_sales(20, 60, num_months),
    "Product_D": generate_random_sales(10, 50, num_months)
}

df_initial = pd.DataFrame(sales_data)
df_initial.to_csv("data/initial.csv", index=False)

# --- 2. Build DataFrame with Computed Metrics ---
df_final = df_initial.copy()

# Ensure Date column is in datetime format
df_final["Date"] = pd.to_datetime(df_final["Date"])

# Set month names as reference keys
df_final["Month"] = df_final["Date"].dt.strftime("%b")

# Product-specific columns list
product_cols = ["Product_A", "Product_B", "Product_C", "Product_D"]

# Compute monthly metrics
df_final["Total_Sales"] = df_final[product_cols].sum(axis=1)
df_final["Average_Sales"] = df_final[product_cols].mean(axis=1)
df_final["Month_over_Month_Growth"] = df_final["Total_Sales"].pct_change() * 100

# Map months to quarters
def assign_quarter(date):
    month = date.month
    if month in [1, 2, 3]:
        return "Q1"
    elif month in [4, 5, 6]:
        return "Q2"
    elif month in [7, 8, 9]:
        return "Q3"
    else:
        return "Q4"

df_final["Quarter"] = df_final["Date"].apply(assign_quarter)

# Identify max and min products for each month
df_final["Max_Sales_Product"] = df_final[product_cols].idxmax(axis=1)
df_final["Min_Sales_Product"] = df_final[product_cols].idxmin(axis=1)

# Export enriched dataset
df_final.to_csv("data/final.csv", index=False)

# --- 3. Pivot Tables & Summaries ---
pivot_quarterly = df_final.pivot_table(
    index="Quarter",
    values=product_cols + ["Total_Sales"],
    aggfunc="mean"
)
pivot_quarterly.to_csv("data/output.csv")

# --- 4. Extract Key Insights ---
best_month_row = df_final.loc[df_final["Total_Sales"].idxmax()]
best_product_name = df_final[product_cols].sum().idxmax()
best_quarter_name = df_final.groupby("Quarter")["Total_Sales"].sum().idxmax()

print("--- Key Operational Insights ---")
print(f"Best Month: {best_month_row['Month']} with Total Sales of {best_month_row['Total_Sales']} units")
print(f"Best Product: {best_product_name} (highest total units sold)")
print(f"Best Quarter: {best_quarter_name} (highest performance across quarters)\n")

# --- 5. Data Visualizations ---
# Set plotting style
sns.set_theme(style="whitegrid")
fig, axs = plt.subplots(2, 2, figsize=(16, 12))

# Plot 1: Line Chart - Product Sales Trends
for col in product_cols:
    axs[0, 0].plot(df_final["Month"], df_final[col], marker='o', label=col.replace("_", " "))
axs[0, 0].set_title("Product Sales Trends Across Months", fontsize=12, fontweight='bold')
axs[0, 0].set_xlabel("Month")
axs[0, 0].set_ylabel("Units Sold")
axs[0, 0].legend()

# Plot 2: Stacked Bar Chart - Monthly Sales Distribution
df_final.set_index("Month")[product_cols].plot(kind="bar", stacked=True, ax=axs[0, 1], colormap="viridis")
axs[0, 1].set_title("Monthly Total Sales Distribution", fontsize=12, fontweight='bold')
axs[0, 1].set_xlabel("Month")
axs[0, 1].set_ylabel("Total Units Sold")
axs[0, 1].legend(title="Products")

# Highlight best month on the stacked bar chart
best_month_index = df_final[df_final["Month"] == best_month_row["Month"]].index[0]
axs[0, 1].annotate(
    "Best Month!",
    xy=(best_month_index, best_month_row["Total_Sales"]),
    xytext=(best_month_index - 0.5, best_month_row["Total_Sales"] + 15),
    arrowprops=dict(facecolor='darkred', shrink=0.05, width=1.5, headwidth=6)
)

# Plot 3: Heatmap - Monthly Sales Density
heatmap_data = df_final.set_index("Month")[product_cols]
sns.heatmap(heatmap_data, annot=True, fmt="d", cmap="YlGnBu", ax=axs[1, 0], cbar_kws={'label': 'Units Sold'})
axs[1, 0].set_title("Monthly Sales Density Heatmap", fontsize=12, fontweight='bold')
axs[1, 0].set_xlabel("Products")
axs[1, 0].set_ylabel("Month")

# Plot 4: Boxplot - Product Sales Variance
sns.boxplot(data=df_final[product_cols], ax=axs[1, 1], palette="Set2")
axs[1, 1].set_title("Product Sales Variance Comparison", fontsize=12, fontweight='bold')
axs[1, 1].set_xlabel("Products")
axs[1, 1].set_ylabel("Units Sold Distribution")
axs[1, 1].set_xticklabels([col.replace("_", " ") for col in product_cols])

plt.tight_layout()
plt.show()
```
