
## 4.5. Tutorial 4. Hospital Pandas and Visualization Solutions

### Problem Statement
Load and analyze hospital patient data using Pandas. The dataset contains records for patients in three cities across various departments.

#### Dataset

| City | Department | Age | DaysAdmitted | DailyCost | Satisfaction | Readmitted |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| Alger | Cardiology | 45 | 5 | 200 | 4.5 | 0 |
| Annaba | Neurology | 60 | 8 | 300 | 3.8 | 1 |
| Oran | Orthopedics | 30 | 3 | 150 | 4.2 | 0 |
| Alger | Cardiology | 50 | 7 | 220 | 4.0 | 0 |
| Oran | Neurology | 40 | 4 | 280 | 4.1 | 1 |
| Annaba | Orthopedics | 70 | 6 | 160 | 3.5 | 1 |
| Alger | Cardiology | 35 | 2 | 210 | 4.7 | 0 |
| Oran | Neurology | 55 | 9 | 290 | 3.9 | 1 |

#### Tasks
1. **Initialize DataFrame**: Load the raw data into a Pandas DataFrame.
2. **Add Engineered Columns**:
   * Calculate `TotalCost` = `DaysAdmitted` * `DailyCost`.
   * Classify patients into an `AgeGroup` column: 'Young' (< 30), 'Middle' (30–50), and 'Senior' (> 50).
3. **Modify Columns**:
   * Add a `Bonus` column (representing a 10% daily cost increase).
   * Rename column `DailyCost` to `DailyFee`.
   * Delete the `Bonus` column.
4. **Data Transformations**:
   * Classify costs with `HighCost` = 'Yes' (if `TotalCost` > 1200), otherwise 'No'.
   * Classify satisfaction with `SatisfactionLevel` = 'Excellent' (>= 4.5), 'Average' (3.5 to 4.5), and 'Poor' (< 3.5).
5. **Add Ranking and Sorting**:
   * Add a column ranking patients by `TotalCost` (highest cost = rank 1).
   * Sort the dataset by the calculated `RiskScore` (defined below) in descending order.
6. **Aggregate Summaries**: Compute the average total cost, average satisfaction, and readmission rate for each city.
7. **Calculate Risk Score**:
   $$\text{RiskScore} = (\text{DaysAdmitted} \times 0.4) + (\text{Age} \times 0.3) + (100 - \text{Satisfaction} \times 10) \times 0.3$$
   Classify scores into `RiskCategory`: 'High' (> 60), 'Medium' (40–60), and 'Low' (otherwise).
8. Identify the city with the highest average risk score.
9. **Clean & Map Data**:
   * Replace missing values in `Satisfaction` with the column mean.
   * Map `City` names to region codes: Alger $\rightarrow$ DZ-16, Annaba $\rightarrow$ DZ-23, Oran $\rightarrow$ DZ-31.
   * Map binary readmission status: 0 $\rightarrow$ 'No', 1 $\rightarrow$ 'Yes'.
10. **Analyze & Export**:
    * Create a pivot table showing the average cost and satisfaction for each city and department combination.
    * Export the processed data to `hospital_risk.csv` and the summary statistics to `city_summary.csv`.

---

### Complete Code Solution

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns

# 1. Initialize DataFrame
data = {
    'City': ['Alger', 'Annaba', 'Oran', 'Alger', 'Oran', 'Annaba', 'Alger', 'Oran'],
    'Department': ['Cardiology', 'Neurology', 'Orthopedics', 'Cardiology', 'Neurology', 'Orthopedics', 'Cardiology', 'Neurology'],
    'Age': [45, 60, 30, 50, 40, 70, 35, 55],
    'DaysAdmitted': [5, 8, 3, 7, 4, 6, 2, 9],
    'DailyCost': [200, 300, 150, 220, 280, 160, 210, 290],
    'Satisfaction': [4.5, 3.8, 4.2, 4.0, 4.1, 3.5, 4.7, 3.9],
    'Readmitted': [0, 1, 0, 0, 1, 1, 0, 1]
}
df = pd.DataFrame(data)

# 2. Add Engineered Columns
df['TotalCost'] = df['DaysAdmitted'] * df['DailyCost']

# Define a function to segment ages
def segment_age(age):
    if age < 30:
        return 'Young'
    elif 30 <= age <= 50:
        return 'Middle'
    else:
        return 'Senior'

df['AgeGroup'] = df['Age'].apply(segment_age)

# 3. Modify Columns
df['Bonus'] = df['DailyCost'] * 0.10
df.rename(columns={'DailyCost': 'DailyFee'}, inplace=True)
df.drop(columns=['Bonus'], inplace=True)

# 4. Data Transformations
# Calculate HighCost status (threshold: 1200)
df['HighCost'] = np.where(df['TotalCost'] > 1200, 'Yes', 'No')

# Define satisfaction level mapping
def get_satisfaction_level(score):
    if score >= 4.5:
        return 'Excellent'
    elif 3.5 <= score < 4.5:
        return 'Average'
    else:
        return 'Poor'

df['SatisfactionLevel'] = df['Satisfaction'].apply(get_satisfaction_level)

# 5. Add Ranking and Sorting
df['CostRank'] = df['TotalCost'].rank(ascending=False, method='min')

# 7. Calculate Risk Score (needed for sorting in step 5)
# Formula: Risk = (DaysAdmitted * 0.4) + (Age * 0.3) + (100 - Satisfaction * 10) * 0.3
df['RiskScore'] = (df['DaysAdmitted'] * 0.4) + (df['Age'] * 0.3) + (100 - df['Satisfaction'] * 10) * 0.3

def segment_risk(score):
    if score > 60:
        return 'High'
    elif 40 <= score <= 60:
        return 'Medium'
    else:
        return 'Low'

df['RiskCategory'] = df['RiskScore'].apply(segment_risk)

# Sort dataset by RiskScore in descending order
df_sorted_risk = df.sort_values(by='RiskScore', ascending=False)

# 6. City-Level Aggregations
city_aggregations = df.groupby('City').agg(
    AvgTotalCost=('TotalCost', 'mean'),
    AvgSatisfaction=('Satisfaction', 'mean'),
    ReadmissionRate=('Readmitted', 'mean')
).reset_index()

# 8. Identify the city with the highest average risk score
highest_risk_city = df.groupby('City')['RiskScore'].mean().idxmax()
print(f"City with the highest average RiskScore: {highest_risk_city}\n")

# 9. Clean & Map Data
# Fill missing satisfaction scores with column mean (none missing in raw data, but included for completeness)
df['Satisfaction'] = df['Satisfaction'].fillna(df['Satisfaction'].mean())

# Map cities to region codes
city_map = {'Alger': 'DZ-16', 'Annaba': 'DZ-23', 'Oran': 'DZ-31'}
df['RegionCode'] = df['City'].map(city_map)

# Map binary readmission status
readmit_map = {0: 'No', 1: 'Yes'}
df['ReadmissionStatus'] = df['Readmitted'].map(readmit_map)

# 10. Pivot Table & Exporting Results
pivot_summary = df.pivot_table(
    index='City',
    columns='Department',
    values=['TotalCost', 'Satisfaction'],
    aggfunc='mean'
)

# Export processed files
df.to_csv("hospital_risk.csv", index=False)
city_aggregations.to_csv("city_summary.csv", index=False)

# Render processed DataFrame
print("--- Processed Hospital Patient DataFrame ---")
print(df[['City', 'RegionCode', 'TotalCost', 'RiskScore', 'RiskCategory', 'ReadmissionStatus']])
```
