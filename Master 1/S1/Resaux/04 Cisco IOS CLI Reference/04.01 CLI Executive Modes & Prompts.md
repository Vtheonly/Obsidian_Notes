### 4.1 CLI Executive Modes & Prompts

Cisco IOS uses a hierarchical command-line structure. Each mode provides access to a specific set of commands.

```mermaid
flowchart TD
    UE["User EXEC<br/>Prompt: Router>"]
    UE -->|enable| PE["Privileged EXEC<br/>Prompt: Router#"]
    PE -->|configure terminal| GC["Global Configuration<br/>Prompt: Router(config)#"]
    GC -->|interface Fa0/0| IC["Interface Configuration<br/>Prompt: Router(config-if)#"]
    GC -->|line vty 0 4| LC["Line Configuration<br/>Prompt: Router(config-line)#"]
    GC -->|router rip| RC["Routing Protocol<br/>Prompt: Router(config-rtr)#"]
```
