
# Reserved Addresses

Think of them as celebrities.

---

## 127.x.x.x

```text
127 = Me
```

Loopback.

The packet never leaves.

It talks to itself.

---

## 255.255.255.255

```text
EVERYBODY!!!
```

Broadcast.

Every device hears it.

---

## Host Bits All Zero

```text
House Number = 0
```

That's the street itself.

Network ID.

Example:

```text
192.168.1.0
```

Means the network.

---

## Host Bits All One

```text
Everybody On This Street
```

Broadcast address.

Example:

```text
192.168.1.255
```

# PART 2: FLSM

FLSM means:

> Everybody gets the same size house.

Think of an apartment building.

Every apartment is identical.

```text
FLSM
=
Equal Subnets
```

---

## The Formula

You only need:

[  
2^n \ge \text{required subnets}  
]

Memory:

```text
Need 8 subnets?

2^3 = 8
```

Need 16?

```text
2^4 = 16
```

Always ask:

> How many bits do I need to create enough subnets?

---

## FLSM Cheat Sheet

```text
Borrow Bits
     ↓
Create Subnets
     ↓
Less Hosts
```

Think:

> More subnet bits = fewer host bits

Like stealing rooms from a house.

---

# Fast FLSM Exam Method



```text

```

Done.

# PART 3: VLSM

FLSM:

```text
Everyone gets same apartment.
```

VLSM:

```text
Everyone gets what they need.
```

---

Imagine:

```text
Family of 50
Family of 50
Family of 30
Family of 12
Family of 2
```

Would you give all of them giant houses?

No.

That's wasteful.

That's why VLSM exists.

---

## Golden Rule

The biggest subnet always comes first.

Remember:

> Big Fish Eat First

Or:

```text
Largest
↓
Medium
↓
Small
↓
Tiny
```

Never do the opposite.

---

## Host Calculation Trick

Memorize:

|Hosts Needed|Prefix|
|---|---|
|2|/30|
|6|/29|
|14|/28|
|30|/27|
|62|/26|
|126|/25|
|254|/24|

This table alone solves most VLSM questions.

---

## Quick Mental Pattern

```text
/24 = 254 hosts
/25 = 126 hosts
/26 = 62 hosts
/27 = 30 hosts
/28 = 14 hosts
/29 = 6 hosts
/30 = 2 hosts
```

Notice:

Hosts roughly halve each step.

# PART 4: Route Summarization

This is the opposite of subnetting.

Subnetting:

```text
One Network
↓
Many Networks
```

Summarization:

```text
Many Networks
↓
One Network
```

---

Memory sentence:

> Subnetting Splits. Summarization Squashes.

---

## Binary Trick

Look for:

```text
Common Left Bits
```

That's literally the entire topic.

Example:

```text
10 = 00001010
11 = 00001011
12 = 00001100
13 = 00001101
14 = 00001110
15 = 00001111
```

What matches?

```text
00001
```

That's all you care about.

Ignore the rest.

---

## Summarization Recipe

Remember:

```text
Binary
Match
Count
Mask
```

or

```text
B M C M
```

Pronounce:

> "Be My Classmate, Mate."

# PART 5: Gateway

This is the easiest section.

Think:

```text
Gateway = Exit Door
```

---

A house cannot use an exit door located in another neighborhood.

Therefore:

> Gateway must be inside the same subnet.

Always.

---

## Exam Shortcut

Whenever you see:

```text
Host IP
Gateway IP
```

Immediately ask:

```text
Same subnet?
```

If:

```text
YES
```

Valid.

If:

```text
NO
```

Invalid.
